    Section Open Sets

    Definition 134. Open Set.
    A set \(U \subseteq \mathbb{R}\) is called an open set if for every \(x \in U\) there is an \(\epsilon > 0\) for which \(V_{\epsilon}(x_0) \subset U\text{.}\)
    🔗
    🔗
    Example 135. Open Interval.
    The open interval \((a,b)\) is an open set.
    🔗
    🔗
    🔗
    Example 136. The Reals.
    The set of real numbers is an open set.
    🔗
    🔗
    🔗
    Example 137. Empty Set.
    The empty set is an open set.
    🔗
    🔗
    🔗
    🔗

    Example 138. Finite Union of Open Sets.
    Any finite union of open sets is open.

    Any finite intersection of open sets is open.

    Section Closed Sets

    Definition 142. Limit Point.
    A point \(x\) is a limit point of a set \(A \subset \mathbb{R}\) if every \(\epsilon\)-neighborhood \(V_{\epsilon}(x)\) intersects \(A\) at some point other than \(x\text{.}\)
    🔗
    🔗
    Example 143. Limit Points of the empty set.
    The empty set has no limit points.
    🔗
    🔗
    Example 144. Limit Points of the set of all reals, \(\mathbb{R}\text{.}\).
    Every real number is a limit point of \(\mathbb{R}\text{.}\)
    🔗
    🔗
    Example 145. Limit Points of the set \(\{0\}\text{.}\).
    The singleton set has no limit points.
    🔗
    🔗
    Example 146. Limit Points of a \(\{0, 1, 2\}\text{.}\).
    The finite set has no limit points.
    🔗
    🔗
    🔗

    Section Closed Sets

    Definition 147. Limit Point.
    A point \(x\) is a limit point of a set \(A \subset \mathbb{R}\) if every \(\epsilon\)-neighborhood \(V_{\epsilon}(x)\) intersects \(A\) at some point other than \(x\text{.}\)
    🔗
    🔗
    Example 148. Terms of the sequence \((a_n)\) where \(a_n=n^{-1}\text{.}\).
    The only limit points of this set is \(a=0\text{.}\)
    🔗
    🔗
    Example 149. \(\{ x \in \mathbb{Q} \, : \, 0 < x < 1\}\).
    The set of limit points is \([0,1] \subset \mathbb{R}\text{.}\)
    🔗
    🔗
    🔗

    Section Closed Sets

    Theorem 150.
    A point \(x\) is a limit point of a set \(A \subset \mathbb{R}\) if and only if \(x=\lim a_n\) for some sequence \((a_n)\) contained in \(A\) with \(a_n \neq x\) for all \(n \in \mathbb{N}\text{.}\)
    🔗
    🔗
    Proof.
    🔗
    🔗
    Definition 151. Isolated Point.
    A point \(x\) is an isolated point of a set \(A \subset \mathbb{R}\) if it is not a limit point of \(A\text{.}\)
    🔗
    🔗
    Definition 152. Closed Set.
    A set \(F \subseteq \mathbb{R}\) is called a closed set if it contains all its limit points.
    🔗
    🔗

    Section Closed Sets

    Definition 158. Closed Set.
    A set \(F \subseteq \mathbb{R}\) is called a closed set if it contains all its limit points.
    🔗
    🔗
    Example 159. Terms of the sequence \((a_n)\) where \(a_n=n^{-1}\text{.}\).
    The set is not closed.
    🔗
    🔗
    Example 160. \(\{ x \in \mathbb{Q} \, : \, 0 < x < 1\}\).
    The set is not closed.
    🔗

    Section Closure

    Definition 161.
    Let \(A \subseteq \mathbb{R}\text{.}\) If \(L\) is the set of limit points of \(A\text{,}\) then the closure of \(A\) is \(\overline{A} = A \cup L\text{.}\)
    🔗
    🔗
    🔗

    Theorem 162.

    If \(A \subseteq \mathbb{R}\text{,}\) then \(\overline{A}\) is a closed set.
    🔗
    🔗
    The set \(\overline{A}\) is the smallest closed set that contains \(A\text{.}\)
