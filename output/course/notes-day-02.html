    Section Number systems: the numbers

    The set of natural numbers (denoted by \(\mathbb{N}\)) forms the most elementary and familiar system of numbers in mathematics.
    🔗
    The set of integers (denoted by \(\mathbb{Z}\text{,}\) from the German word for number, Zahlen) augments the naturals with zero and negative counting numbers.
    🔗
    The set of rational numbers (denoted by \(\mathbb{Q}\text{,}\) for quotient) adds lots of fractional numbers.
    🔗
    The Greeks get credit for being the first society that investigated mathematics rigorously. They believed "all is number" and by "number" they mean rational numbers.
    🔗
    🔗

    Section Number systems:the operations

    What are the basic operations endowed to \(\mathbb{N}\text{,}\) \(\mathbb{Z}\text{,}\) and \(\mathbb{Q}\text{?}\)
    🔗
    The sets are also ordered. What does this mean?
    🔗
    Without getting too deep into the concept of algebraic structure of these systems, all these systems are called ordered commutative rings because
    • they have (and are closed under) two operations: addition and multiplication
      🔗
    • both two operations are commutative and associative
      🔗
    • multiplication is distributive over addition
      🔗
    • addition has an inverse
      🔗
    • and both operations have an identity
      🔗
    🔗
    The rationals is also an ordered commutative ring, but it has a bit more ’structure’ because everyt element has a multiplicative inverse. This means that the rational numbers form a field.
    🔗
    The Greeks believed every number was rational.
    🔗
    They were wrong.
    🔗
    🔗

    Section Irrational numbers

    Theorem 4.
    The number \(\sqrt{2}\) is an irrational number.
    🔗
    🔗
    Proof.
    We prove this by contradiction or indirectly. Suppose \(\sqrt{2}\) is not irrational. This means that the number is rational. This means that \(\sqrt{2}\) can be expressed as
    \begin{gather*} \sqrt{2}=\frac{p}{q} \end{gather*}
    where \(p\) and \(q\) are natural numbers.
    🔗
    Without loss of generality, we can assume \(p/q\) is a proper fraction (the numerator and denominator have no prime factors in common).
    🔗
    Then
    \begin{gather*} \sqrt{2}^2=\left( \frac{p}{q} \right)^2 = \frac{p^2}{q^2} \end{gather*}
    so
    \begin{gather*} 2q^2=p^2\text{.} \end{gather*}
    🔗
    The left-hand side of this equation is an even number, so \(p^2\) is even, too. This means 2 is a prime factor of \(p^2\text{.}\) But all prime factrors of \(p^2\) appear at least twice, so we know \(p\) is event. This means \(p=2r\) for some natural number \(r\text{.}\)
    🔗
    This allows us to write
    \begin{gather*} 2q^2=p^2=(2r)^2=4r^2\text{.} \end{gather*}
    Diving both sides by 2 gives
    \begin{gather*} q^2=2r^2\text{.} \end{gather*}
    This shows that \(q\) must be even, too.
    🔗
    We already showed that \(p\) was even, and we began this proof by assuming that \(p\) and \(q\) had no common prime factors. So \(q\) cannot be even. This is our contradiction.
    🔗
    Consequently, we deduce that \(\sqrt{2}\) cannot be rational. It must be irrational, as originally claimed.
    🔗
    🔗
    🔗

    Section Constructing the Real Number system

    Mathematician knew they needed a number system that formed a continuum (i.e., has no holes) and shares the algebraic properties of the rational number field.
    🔗
    To this point in our mathematical education, we’ve worked with the real numbers withot knowing much about what they are and where they came from.
    🔗
    In this course, we will construct them, starting from the rational numbers, as the Greeks would have wanted.
    🔗

    Section Sets and Operations

    (I expect the following to be review from Transitions and Discrete.)
    🔗
    A set is a collection of elements.
    🔗
    Let \(A\) be a set.
    🔗
    If \(x\) is an element of \(A\text{,}\) we write \(x \in A\)
    🔗
    If \(x\) is not an element of \(A\text{,}\) we write \(x \notin A\)
    🔗
    There are two main kinds of set notation: roster notation and set-builder notation.
    🔗
    A set is represented in roster notation if its elements are written out, separated by commas, and this list is wrapped in curley braces, { and }. When we represent a set in set-builder notation we use curley braces as before, but the elements of the set are described differently; the notation looks like \(\{ \text{element } | \text{ rule} \}\) or \(\{ \text{element } : \text{ rule} \}\) where ’element’ is a symbolic representation of the set elements, and ’rule’ is some pattern that the elements follow. This is best described with some examples.
    🔗
    Example 5. A Set and Elements.
    If \(A=\{1,2,3\}\) then \(1 \in A\) and \(4 \notin A\text{.}\)
    🔗
    🔗
    Example 6. Set of all even numbers.
    The set of all even numbers in roster notation would be \(\{ 2, 4, 6, 8, ... \}\text{.}\) In set-builder notation we could write this as
    \begin{equation*} \{2*n \, | \, n \text{ is a natural number}\}\text{.} \end{equation*}
    🔗
    🔗
    Example 7. An infinite solution set.
    The set of solutions to \(cos(x)=1\) is
    \begin{equation*} \{\pi/2+2\pi*k \, | \, k \text{ is an integer} \}\text{.} \end{equation*}
    In roster notation this might look like
    \begin{equation*} \{... \pi/2 -4\pi,\pi/2 -2\pi, \pi/2, \pi/2 +2\pi, \pi/2 +4\pi, ...\}. \end{equation*}
    🔗
    🔗
    🔗

    Definition 8.

    The set of real numbers strictly between \(a, b in mathbb{R}\) is called an open interval from \(a\) to \(b\) and is denoted by \((a,b)\text{.}\) Note that
    \begin{gather*} (a,b) = \{ x \in \mathbb{R} \, : \, a < x < b \}\text{.} \end{gather*}
    🔗
    The set of real numbers between \(a, b in mathbb{R}\) inclusive is called a closed interval from \(a\) to \(b\) and is denoted by \([a,b]\text{.}\) Note that
    \begin{gather*} [a,b] = \{ x \in \mathbb{R} \, : \, a \leq x \leq b \}\text{.} \end{gather*}
    🔗
    🔗

    Section

    There are three important operations on sets: union, intersection, and complement.
    🔗
    Definition 9. Set Union.
    If \(A\) and \(B\) are sets, the union of \(A\) and \(B\) is denoted by
    \begin{gather*} A \cup B \end{gather*}
    and is the set consisting of elements that are either in set \(A\) or in set \(B\text{.}\)
    🔗
    🔗
    Example 10.
    If \(A=\{1,2,3\}\) and if \(B=\{3,4,5\}\text{,}\) then
    \begin{gather*} A \cup B = \{1,2,3,4,5\}\text{.} \end{gather*}
    🔗
    🔗
    Definition 11. Set Intersection.
    If \(A\) and \(B\) are sets, the intersection of \(A\) and \(B\) is denoted by
    \begin{gather*} A \cap B \end{gather*}
    and is the set consisting of elements that are in both set \(A\) and in set \(B\text{.}\)
    🔗
    🔗
    Example 12.
    If \(A=\{1,2,3\}\) and if \(B=\{3,4,5\}\text{,}\) then
    \begin{gather*} A \cap B = \{3\}\text{.} \end{gather*}
    🔗
    🔗
    Definition 13. Subset.
    The set \(A\) is a subset of another set \(B\) if every element in \(A\) is also in \(B\text{.}\) If \(A\) is a subset of \(B\) we write
    \begin{gather*} A \subseteq B\text{.} \end{gather*}
    🔗
    🔗
    Example 14.
    If \(E\) is the set of even numbers, then \(E \subseteq \mathbb{N}\text{.}\)
    🔗
    🔗
    🔗

    Section

    Definition 15. Set Complement.
    The complement of a set \(A\) is relative to another set \(B\) that contains \(A\text{,}\) i.e., \(A \subseteq B\text{.}\)
    🔗
    If \(A \subseteq B\text{,}\) then the complement of \(A\text{,}\) denoted by \(A^c\text{,}\) i the set of elements in \(B\) that are not in \(A\text{.}\)
    🔗
    🔗
    Example 16.
    In \(\mathbb{Z}\text{,}\) the complement of the positive numbers is the non-positive numbers, \(\{0,-1, -2, -3, \ldots\}\text{.}\)
    🔗
    🔗
    Example 17.
    In \(\mathbb{N}\text{,}\) the complement of the odd numbers is the even numbers.
    🔗
    🔗
    Example 18.
    In \(\mathbb{R}\text{,}\) the complement of \(\mathbb{Q}\) is the irrational numbers.
    🔗
    🔗
    🔗

    Section

    If we have multiple sets, \(A_1\text{,}\) \(A_2\text{,}\) \(A_3, \ldots\text{,}\) \(A_n\text{,}\) we can take the union of all the sets or the intersection of all the sets. Special notation for these are (respectively)
    \begin{gather*} A_1 \cup A_2 \cup A_3 \cup \ldots \cup A_n = \bigcup_{i=1}^n A_i\\ A_1 \cap A_2 \cap A_3 \cap \ldots \cap A_n = \bigcap_{i=1}^n A_i\text{.} \end{gather*}
    🔗
    Union, intersection, and complement come together in DeMorgan’s Law.
    🔗
    Theorem 19. DeMorgan’s Law.
    If \(A_1\text{,}\) \(A_2\text{,}\) \(A_3, \ldots\text{,}\) \(A_n\) are sets, then
    \begin{gather*} \left( \bigcup_{i=1}^n A_i \right)^c=\bigcap_{i=1}^n A_i^c, \text{ and }\\ \left( \bigcap_{i=1}^n A_i \right)^c=\bigcup_{i=1}^n A_i^c\text{.} \end{gather*}
    In fact, the equality holds for infinite collections of sets, too.
    🔗
    You should be able to prove this theorem.
    🔗
    🔗

    Section Bounded Sets

    Definition 20.
    A set \(A \subseteq \mathbb{R}\) is bounded above if there is an \(M \in \mathbb{R}\) for which \(M > a\) for all \(a \in A\text{.}\)
    🔗
    A set \(A \subseteq \mathbb{R}\) is bounded below if there is an \(m \in \mathbb{R}\) for which \(m < a\) for all \(a \in A\text{.}\)
    🔗
    A set \(A \subseteq \mathbb{R}\) is totally bounded if there is an \(K \in \mathbb{R}\) for which \(|a| < K\) for all \(a \in A\text{.}\)
    🔗
    🔗
    Example 21.
    Each of the following sets in bounded in one way or another. For each, find an upper bound and a lower bound and a total bound, if possible.
    🔗
    1. \(\displaystyle [0,1]\)
      🔗
    2. \(\displaystyle (-\infty,3)\)
      🔗
    3. \(\displaystyle [5,\infty)\)
      🔗
    4. \(\displaystyle [-3,1/3)\)
      🔗
