    Section Note: index sets

    When we have a collection of things, we often organize them using an index. This is especially helpful when we need to order or manipulate elements of the collection.
    🔗
    Example 182. Finite sum.
    Consider this finite collection of numbers \(A=\{a_1, a_2,a_3,\ldots,a_N\}\text{.}\) Here we are using subscripts to index the numbers in this collection. Let \(I=\{1, 2, 3,\ldots,N\}\text{.}\) This is the index set we are using to index our collection of numbers. We can use it to express \(A\) using the index set \(I\) as follows:
    \begin{gather*} A=\{a_i \, : \, i \in I\}. \end{gather*}
    🔗
    When we learned about sigma-notation in Calculus 1, we learned to write the sum of the numbers in \(A\) as
    \begin{gather*} \displaystyle \sum_{i=1}^N a_i. \end{gather*}
    In this notation we actually call \(i\) the index of summation. We can write the same sum using the index set \(I\) as follows:
    \begin{gather*} \displaystyle \sum_{i=1}^N a_i = \sum_{i \in I} a_i. \end{gather*}
    🔗
    🔗
    🔗

    Section Note: index sets

    Example 183. Series.
    Consider the infinite collection of numbers \(A=\{a_1, a_2,a_3,\ldots,\}\text{.}\) Here we are using subscripts to index the numbers in this collection, and the index set is \(\mathbb{N}\text{.}\) We can use the index to express \(A\) as follows:
    \begin{gather*} A=\{a_i \, : \, i \in \mathbb{N}\}. \end{gather*}
    🔗
    When we learned about sigma-notation in Calculus 1, we learned to write the infinite sum of the numbers in \(A\) as
    \begin{gather*} \displaystyle \sum_{i=1}^{\infty} a_i. \end{gather*}
    We can write the same sum using the index set \(I\) as follows:
    \begin{gather*} \displaystyle \sum_{i=1}^{\infty} a_i = \sum_{i \in \mathbb{N}} a_i. \end{gather*}
    🔗
    🔗
    Example 184. Collections.
    Suppose we have a mathematical object \(X_{\lambda}\text{.}\) (This object cold be a number, or a set, or a function, or something else.). Here \(\lambda\) is going to serve as an index. Let \(\Lambda\) be our index set. Then the set (or collection) of all our objects can be written
    \begin{gather*} X = \{ x_{\lambda} \, : \, \lambda \in \Lambda\}. \end{gather*}
    🔗
    If the objects are numbers, we could write their sum as
    \begin{gather*} \sum_{\lambda \in \Lambda} X_{\lambda}. \end{gather*}
    🔗
    If the objects are sets, we could write their union as
    \begin{gather*} \displaystyle \bigcup_{\lambda \in \Lambda} X_{\lambda}. \end{gather*}
    and their intersection as
    \begin{gather*} \displaystyle \bigcap_{\lambda \in \Lambda} X_{\lambda}. \end{gather*}
    🔗
    And so on.
    🔗

    Section Compactness

    Definition 185.
    A set \(K \subset \mathbb{R}\) is (sequentially) compact if every sequences in \(K\) has a subsequence that converges in \(K\text{.}\)
    🔗
    🔗
    Example 186. \([0,1]\) is compact.
    Consider \(K=[0,1] \subset \mathbb{R}\text{.}\) We want to show that \(K\) is compact.
    🔗
    Let \((x_n) \subset K\) be a sequence. We will show that \((x_n)\) has a subsequence \((x_{n_k})\) that converges in \(K\text{.}\) That means \((x_{n_k}) \rightarrow x_0 \in K\text{.}\)
    🔗
    Since \(0 < x_n < 1\) for all \(n \in \mathbb{N}\text{,}\) the sequence \((x_n)\) is bounded. The Bolzano-Weierstrass Theorem tells us that \((x_n)\) has a subsequence \((x_{n_k})\) that converges in \(\mathbb{R}\text{.}\) Let\(x_0\) be the limit point of that subsequence. Since \(K\) is closed, it contains all its limit points. By Theorem 3.2.5, \(x_0\) is a limit point of \(K\text{.}\) Therefore, \(x_0 \in K\text{.}\) This shows that \([0,1]\) is compact.
    🔗
    🔗
    Example 187. \([a,b]\) is compact.
    For exact same reason, every closed and bonded interval is compact.
    🔗
    🔗
    Recall that a set \(A \subset \mathbb{R}\) is bounded if there is an \(M \in \mathbb{R}\) for which \(-M \leq x \leq M\) for all \(x \in A\text{.}\)
    🔗
    🔗

    Section Compactness

    Theorem 188.
    A set\(K \subset \mathbb{R}\) is (sequentially) compact if and only if \(K\) is closed and bounded.
    🔗
    🔗
    Proof.
    (\(\Rightarrow\)). Suppose\(K\) is compact. We must show that \(K\) is closed and bounded.
    🔗
    Suppose that \(K\) is not bounded. Then for each \(n \in \mathbb{N}\) there is an \(x_n \in K\) with \(|x_n| > n\text{.}\) These elements of \(K\) form a sequence \((x_n) \subset K\text{.}\) Since \(K\) is (sequentially) compact, the sequence has a subsequence \((x_{n_k})\) that converges in \(K\text{.}\) But \(|x_{n_k}| > n_k \rightarrow \infty\) as \(k \rightarrow \infty\text{.}\) This means that the subsequence is unbounded. However, we’ve proved that convergent sequences must be bounded. This is a contradiction. Therefore, \(K\) must be bounded.
    🔗
    To see that \(K\) is closed, let \(x \in \mathbb{R}\) be a limit point of \(K\text{.}\) We must show that \(x \in K\text{.}\) Since \(x\) is a limit point of \(K \text{,}\) there is a sequence \((y_k) \subset K\) that converges to \(x\text{.}\) Since \(K\) is (sequentially) compact, \((y_k)\) has a subsequence \((y_{k_{\ell}})\) that converges in \(K\text{.}\) We know that a subsequence of a convergent sequence must converge to the same limit at the sequence, so \((y_{k_{\ell}}) \rightarrow x\text{,}\) and \(x \in K\text{.}\) This shows that\(K\) is closed.
    🔗
    This completes the proof in this direction.
    🔗
    (\(\Leftarrow\)). Suppose\(K\) is closed and bounded. We must show \(K\) is compact. Let \((x_n) \subset K\) be a sequence. We must show that \((x_n)\) has a subsequence that converges to a point in \(K\text{.}\)
    🔗
    Because \(K\) is bounded, there is an \(M \in \mathbb{R}\) with \(M > 0\) such that \(|x| < M\) for all \(x \in K\text{.}\) Therefore, \(|x_n| < M\) for all \(n \in \mathbb{N}\text{.}\) This means that \((x_n)\) is a bonded sequence. The Bolzao-Weierstrass Theorem implies that \((x_n)\) has a convergent subsequence \((x_{n_k})\text{.}\) Since \((x_{n_k}) \subset K\) and \(K\) is closed (it contains all its limit points), \(\displaystyle \lim_{k \rightarrow \infty} (x_{n_k}) \in K\text{.}\) This shows that that \(K\) is compact.
    🔗
    This completes the proof.
    🔗
    🔗
    🔗

    Section Nested Set Property

    Theorem 189. Nested Set Property.
    If \(K_i\) is a nonempty (sequentially) compact set, and if \(K_i \supseteq K_{i+1}\) for all \(i \in \mathbb{N}\) then \(\displaystyle \cap_{i=1}^{\infty} K_i\) is nonempty.
    🔗
    🔗
    This statement should look familiar.
    🔗
    This is a generalization of the Nested Interval Property that replaces intervals with compact sets.
    🔗
    Proof.
    For each \(n \in\mathbb{N}\text{,}\) because \(K_n\) is nonempty, we can find an element in \(K_n\text{,}\) call it \(x_n\text{.}\) This creates a sequence \((x_n) \subset \mathbb{R}\text{.}\) Since \(K_n \subseteq K_1\) for all \(n\text{,}\) \((x_n) \subset K_1\text{.}\) Because \(K_1\) is a (sequentially) compact set, \((x_n) \subset K_1\) has a subsequence \((x_{n_{\ell}}) \subset K_1\) that converges in \(K_1\text{.}\) Call its limit \(x \in K_1\text{.}\)
    🔗
    Notice that, given any \(m \in \mathbb{N}\text{,}\) \(K_m\) contains the tail of \((x_n)\text{.}\) Specifically, \((x_n)_{n=m}^{\infty} \subset K_m\text{.}\) Since \(K_m\) is compact, the limit of \((x_n)\) is in \(K_m\text{,}\) i.e., \(x \in K_m\text{.}\) It follows that \(\displaystyle x \in \bigcap_{m=1}^{\infty} K_m \text{.}\)
    🔗
    This completes the proof.
    🔗

    Section Open Covers

    Let \(A \subseteq \mathbb{R}\text{.}\) Let \(U_i \subset \mathbb{R}\) be open for \(i \in I\) where \(I\) is some index set. We say the collection \(\mathcal{U}=\{U_i \, : \, i \in I\}\) is an open cover for \(A\) if
    \begin{gather*} \displaystyle A \subseteq \bigcup_{i \in I} U_i \end{gather*}
    If \(I_0 \subseteq I\) is a finite subset of the inde \(I\) for which
    \begin{gather*} \displaystyle A \subseteq \bigcup_{i \in I_0} U_i \end{gather*}
    we call \(\{I_i \, :\, i \in I_0\}\) a finite subcover of \(A\text{.}\)
    🔗
    A set \(A \subseteq \mathbb{R}\) can have many possible open covers. Here are some examples.
    🔗
    Example 190. An open cover of \([a,b]\).
    Consider the set \(U=(a-1,b+1)\text{.}\) The collection \(\mathcal{U}=\{U\}\) is an open over because
    \begin{align*} \displaystyle \bigcup_{U \in \mathcal{U}} U \amp = U =(a-1,b+1) \supset [a,b]. \end{align*}
    🔗
    🔗
    Example 191. An open cover of \([a,b]\).
    Let \(\epsilon > 0\) be given. For \(x \in [a,b]\) define \(U_x = V_{\epsilon}(x)\text{.}\) Then \(\mathcal{U}=\{U_x \, : \, x \in [a,b]\}\) is an open cover of \([a,b]\) because
    \begin{align*} \displaystyle \bigcup_{x \in [a,b]} U_x \amp = \bigcup_{x \in [a,b]} V_{\epsilon}(x) = (a-\epsilon,b+\epsilon) \supset [a,b] \end{align*}
    🔗
    🔗
    🔗

    Section Heine-Borel

    Example 192. An open cover of \([0,1]\) with a finite subcover.
    Consider \(V_{1/2}(x)\) and \(\mathcal{U} = \{ V_{1/2}(x) \, : \, x \in [0,1]\}\text{.}\) This is an open cover of \([0,1]\text{.}\)
    🔗
    One finite subcover of \([0,1]\) is
    \begin{gather*} \{V_{1/2}(1/4), V_{1/2}(1/2), V_{1/2}(3/4)\} \end{gather*}
    🔗
    🔗
    Theorem 193. Heine-Borel.
    A set \(K \subset \mathbb{R}\) is compact if and only if every open cover of \(K\) contains a finite subcover.
    🔗
    🔗
    Note our textbook’s Example 3.3.7.
    🔗
    Proof.
    (\(\Rightarrow\)) Suppose \(K\) is (sequentially) compact. Let \(\{U_i\}\) be a countable open over of\(K\text{.}\) We claim that there is an \(n \in \mathbb{N}\) for which \(\{U_i \, : \, i=1, 2, 3,\ldots,n\}\) is a subcover of \(K\text{.}\)
    🔗
    Suppose there is no such \(n \in \mathbb{N}\text{.}\) Then \(\{U_i \, : \, i=1, 2, 3,\ldots,n\}\) is not a cover for any value of \(n \in \mathbb{N}\text{.}\)
    🔗
    Because
    \begin{gather*} U_1 \subseteq U_1 \cup U_2 \subseteq U_1 \cup U_2 \cup U_3 \subseteq \cdots \subseteq \bigcup_{i=1}^N U_i \end{gather*}
    the successive unions of the open sets in the open cover form a nested sequence of open sets. This means that
    \begin{gather*} K \setminus U_1 \supseteq K \setminus (U_1 \cup U_2) \supseteq K \setminus (U_1 \cup U_2 \cup U_3) \supseteq \cdots \supseteq K \setminus \large( \bigcup_{i=1}^N U_i \large). \end{gather*}
    🔗
    Since \(\{U_i \, : \, i=1, 2, 3,\ldots,n\}\) is not a cover for \(K\text{,}\) the set \(K \setminus \bigcup_{i=1}^n U_i\) is not empty. Choose \(x_n \in K \setminus \bigcup_{i=1}^n U_i\text{.}\) Doing this for each \(n \in\mathbb{N}\) creates a sequence \((x_n) \subset K\text{.}\)
    🔗
    🔗
    🔗

    Since \(U_m\) is open, there in an \(\epsilon > 0\) for which \(V_{\epsilon}(x) \subseteq U_m\text{.}\) Since \((x_{n_k}) ]rightarrow x\text{,}\) there is an \(N \in \mathbb{N}\) for which \(k > N\) implies that \(|x_{n_k} - x| < \epsilon\text{.}\) That is to say, if \(k > N\text{,}\) when \(x_{n_k} \in V_{\epsilon}(x)\text{.}\)

    (\(\Leftarrow\)) Suppose \(K\) has the property that every open cover of \(K\) has a finite subcover. This is called the Heine-Borel property. We want to show that \(K\) is sequentially compact. We will do this by showing that \(K\) is closed and bounded.
    🔗
    To show that \(K\) is bonded, consider the open sets
    \begin{align*} V_1(x)\amp =\{y \in \mathbb{R}\, : \, |x-y| < 1\} \end{align*}
    These are the \(1-\)neighborhoods of \(x\text{.}\) Then the collection \(\{v_1(x)\}_{x \in K}\) is an open cover of \(K\text{.}\) Since \(K\) has the Heine-Borel property, this open cover has a finite subcover \(\{V_1(x_k)\}_{k=1}^n\text{,}\) noting that \(x_k \in K\) for all \(k\text{.}\)

    🔗
    Consequently, \(K\) is compact.
