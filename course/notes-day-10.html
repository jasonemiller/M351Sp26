    1. \((c \cdot x_n) \rightarrow c \cdot x_0\) for all \(c \in \mathbb{R},\)

    Section Order Limit Theorems

    Theorem 105.
    Assume \((x_n) \rightarrow x_0\) and \((y_n) \rightarrow y_0\text{.}\)
    1. If \(x_n \geq 0\) for all \(n \in \mathbb{N}\) then \(x_0 \geq 0\text{.}\)
      🔗
    2. If \(x_n \geq y_n\) for all \(n \in \mathbb{N}\) then \(x_0 \geq y_0\text{.}\)
      🔗
    3. If there exists a \(c \in \mathbb{R}\) for which \(c \leq y_n\) for all \(n \in \mathbb{N}\) then \(c \leq y_0\text{.}\) Likewise, if \(x_n \leq c\) for all \(n \in \mathbb{N}\) then \(x_0 \leq c\text{.}\)
      🔗
    🔗

    Section

    Proof.
    Proof of ``If \(x_n \geq y_n\) for all \(n \in \mathbb{N}\) then \(x_0 \geq y_0\text{.}\)’’
    🔗
    By the algebraic limit theorem, the sequence \((x_n-y_n)\) converges to \(x_0-y_0\text{.}\) By the first part of the Order Limit Theorem, since \(x_n-y_n \geq 0\) or all \(n \in \mathbb{N}\text{,}\) we know \(x_0-y_0 \geq 0\text{.}\) Therefore, \(x_0 \geq y_0\text{,}\) as desired.
    🔗
    🔗
    Proof.
    Proof of ``If there exists a \(c \in \mathbb{R}\) for which \(c \leq y_n\) for all \(n \in \mathbb{N}\) then \(c \leq y_0\text{.}\) Likewise, if \(x_n \leq c\) for all \(n \in \mathbb{N}\) then \(x_0 \leq c\text{.}\)’’
    🔗
    Take \(x_n=c\text{,}\) the constant sequence and the result follows by the second part of the Order Limit Theorem. Likewise, take \(y_n=c\) and the second part of the statement follows from the second part of the Order Limit Theorem.

    Section Monotone Sequences

    Definition 106.
    A sequence \((x_n)\) is increasing if \(a_n \leq a_{n+1}\) for all \(n \in \mathbb{N}\) and decreasing if \(a_n \geq a_{n+1}\) for all \(n \in \mathbb{N}\text{.}\) A sequence is monotone if it is either increasing or decreasing.
    🔗
    🔗
    Theorem 107. Monotone Convergence Theorem.
    If a sequence is monotone and bounded, then it converges.
    🔗
    Proof.
    Assume the sequence \((x_n)\) is bounded and monotone. Since it is bounded, it is both bounded above and bonded below.
    🔗
    Consider the case where \((x_n)\) is increasing. Then the set \(A= \{ x_n \, : \, n \in \mathbb{N} \} \subset \mathbb{R}\) is bonded above. By the Axiom of Completeness, \(A\) has a least upper bound \(x_0=\sup(A)\text{.}\) We claim that \(\displaystyle \lim_{n \rightarrow \infty} x_n = x_0\text{.}\)
    🔗
    Let \(\epsilon > 0\) be given. By Lemma 1.3.8, \(x_0-\epsilon\) is not an upper bound for \(A\text{.}\) So there exists a point in the sequence \(x_N\) at which \(x_0-\epsilon < x_N\text{.}\) Since our sequence is increasing, we know that if \(n \geq N\) then \(x_N \leq x_n\text{.}\) Therefore,

    Section Subsequences

    Definition 108.
    Let \((x_m)\) be a sequence of real numbers, and let \(n_1 < n_2 < n_3 < \cdots\) be an increasing sequence of natural numbers. Then the sequence
    \begin{gather*} (x_{n_1}, x_{n_2}, x_{n_3}, x_{n_4}, \ldots) \end{gather*}
    is called a subsequence of \((x_n)\) and is denoted by \((x_{n_k})\text{,}\) where \(k \in \mathbb{N}\) indexews the subsequence.
    🔗
    🔗
    Theorem 109.
    Subsequences of a convergent sequence converge to the same limit as the original sequence.
