    Section Functional Limits

    In Calculus 1, we learn about the concepts of continuity of a function and differentiability of a function. Both these concepts require the use of functional limits.
    \begin{gather*} \displaystyle \lim_{x \rightarrow c} f(x) = L\\ \displaystyle \lim_{x \rightarrow c} f(x) = f(c)\\ f'(x) = \displaystyle \lim_{h \rightarrow 0} \frac{1}{h} (f(x+h)-f(x)) \end{gather*}
    🔗
    The central idea of a functional limit is this:
    \begin{gather*} \displaystyle \lim_{x \rightarrow c} f(x) = L \end{gather*}
    🔗
    Definition 201. Functional Limit.
    Let \(f:A \rightarrow \mathbb{R}\) and let \(c\) be a limit point of the domain \(A\text{.}\) We say that \(\lim_{x \rightarrow c} f(x)=L\) provided that, for all \(\epsilon > 0\text{,}\)there exists a \(\delta > 0\) such that whenever \(0 < |x-c| < \delta\) (and \(x \in A\)) it follows that \(|f(x)-L| < \epsilon\text{.}\)
    🔗
    🔗
    🔗

    Definition 202. Functional Limit: Topological Version.
    Let \(f:A \rightarrow \mathbb{R}\) and let \(c\) be a limit point of the domain \(A\text{.}\) We say that \(\lim_{x \rightarrow c} f(x)=L\) provided that, for every \(\epsilon-\)neighborhood \(V_{\epsilon}(L)\) of \(L\) there exists a \(\delta-\)neighborhood \(V_{\delta}(c)\) of \(c\) with the property that for all \(x \in V_{\delta}(c)\) different from \(c\) (with \(x \in A\)) it follows that \(f(x) \in V_{\epsilon}(L)\text{.}\)

    Section Sequential Criterion

    Theorem 206.
    Given a function \(f: A \rightarrow \mathbb{R}\) and a limit point \(c \in A\text{,}\) the following two statements are equivalent:
    1. \(\displaystyle \displaystyle \lim_{x \rightarrow c} f(x)=L\)
      🔗
      🔗
    2. For all sequences \((x_n) \subset A\) satisfying \(x_n \neq c\) and \((x_n) \rightarrow c\text{,}\) it follows that \(f(x_n) \rightarrow L\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗
    Proof.
    (\(\Rightarrow\)) Suppose the first statement is true. We must show that given any sequence \((x_n) \subset A\) satisfying \(x_n \neq c\) we can conclude that \(f(x_n) \rightarrow L\text{.}\) This means, given an \(\epsilon > 0\text{,}\) we want to be able find an \(N=N(\epsilon)\) for which \(n > N\) implies \(|f(x_n)-L| < \epsilon\text{.}\)
    🔗
    Let \((x_n) \subset A\) be a sequence satisfying \(x_n \neq c\text{.}\) And let \(\epsilon > 0\) be given. Since \(\lim_{x \rightarrow c} f(x)=L\text{,}\) we know there is a \(\delta > 0\) for which \(0 < |x-c| < \delta\) implies that \(|f(x)-L| < \epsilon\text{.}\)
    🔗
    We invoke the definition of sequential convergence of \((x_n)\) with \(\delta > 0\text{.}\) There is an \(N_1=N_1(\delta)\) for which \(n > N\) means \(|x_n-c| < \delta\text{.}\) This lets us invoke the given (1) to conclude that \(|f(x_n)-L| < \epsilon\text{,}\) as desired.
    🔗
    (\(\Leftarrow\)). Suppose that the second statement is true. We must show that, given any \(\epsilon > 0\) we can find a \(\delta > 0\) for which \(|x-c| < \delta\) implies \(|f(x)-L| < \epsilon\text{.}\)
    🔗
    The book proves the contrapositive by contradiction! That is, it proves that if (1) is false then (2) is false by assuming (1) is false and (2) is true, arriving at a contradiction. Here’s how that goes.
    🔗
    Suppose (2) is true. For (1) to be false means there is at least one \(\epsilon_0 > 0\) for which no \(\delta > 0\) gets \(x\) close enough to \(c\) to guarantee \(|f(x)-L| < \epsilon_0\text{.}\) This means there will always be at least one input \(x_*\) with \(|x_*-c| < \delta\) and with \(|f(x_*)-L| > \epsilon_0\text{.}\)
    🔗
    Consider \(\delta_n=1/n\text{.}\) For each \(n\text{,}\) let \(x_n\) be the input with \(|x_n-c| < \delta\) and with \(|f(x_n)-L| > \epsilon_0\text{.}\) This gives us a sequence \((x_n)\) that converges to \(c\) at the same time that the sequence \((f(x_n))\) does not converge to \(L\text{.}\) But this contradicts (2), which we took to be true. Therefore, (1) cannot be false at the same time (2) is true. Therefore (2) must be false. We conclude that (2) implies (1).
    🔗
    🔗
    🔗

    Section Useful Corollaries

    Corollary 207. Algebraic Limit Theorem for Functional Limits.
    Let \(f\) and \(g\) be functions defined on a domain \(A \subset \mathbb{R}\text{,}\) and assume \(\lim_{x \rightarrow c}f(x)=L\) and \(\lim_{x \rightarrow c}g(x)=M\) for some limit point \(c\) of \(A\text{.}\) Then
    1. \(\displaystyle \lim_{x \rightarrow c} kf(x) = kL\) for all \(k \in \mathbb{R}\)
      🔗
      🔗
    2. \(\displaystyle \displaystyle \lim_{x \rightarrow c} (f(x)+g(x)) = L+M\)
      🔗
      🔗
    3. \(\displaystyle \displaystyle \lim_{x \rightarrow c} f(x)g(x) = LM\)
      🔗
      🔗
    4. \(\displaystyle \lim_{x \rightarrow c} f(x)/g(x) = L/M\text{,}\) provided \(M \neq 0\)
      🔗
      🔗
    🔗
    🔗
    Corollary 208. Divergence Criterion for Functional Limits.
    Let \(f\) be a function defined on \(A \subset \mathbb{R}\text{,}\) and let \(c\) be a limit point of \(A\text{.}\) If there exists two sequences \((x_n)\) and \((y_n)\) in \(A\) with \(x_n \neq c\) and \(y_n \neq c\) and
    \begin{align*} \lim x_n \amp = \lim y_n =c \end{align*}
    but
    \begin{align*} \lim f(x_n) \amp \neq \lim f(y_n) \end{align*}
    then we can conclude that the function limit \(\lim_{x \rightarrow c} f(x)\) does not exist.
    🔗
    🔗
    🔗

    Section Continuous Functions

    🔗
    Definition 209. Continuity at a point.
    A function \(f:A \rightarrow \mathbb{R}\) is continuous at a point \(c \in A\) if, for all \(\epsilon > 0\) there exists a \(\delta > 0\) such that whenever \(|x-c| < \delta\) (and \(a \in A\)) it follows that \(|f(x)-f(c)| < \epsilon\text{.}\)
    🔗
    🔗
    Note here that the value of \(\delta\) depends on both \(\epsilon\) and \(c\text{,}\) so we write \(\delta=\delta(\epsilon,c)\text{.}\)
    🔗
    Definition 210. Continuous on a set.
    If \(f: A \rightarrow \mathbb{R}\) is continuous at every point in the domain \(A\) we say \(f\) is continuous on \(A\text{.}\)
