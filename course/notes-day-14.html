    Section Open Sets

    Definition 163. Open Set.
    A set \(U \subseteq \mathbb{R}\) is called an open set if for every \(x \in U\) there is an \(\epsilon > 0\) for which \(V_{\epsilon}(x_0) \subset U\text{.}\)
    🔗
    🔗
    Canonical example of an open set is an open interval.
    🔗
    Other good (but extreme) examples: \(\mathbb{R}\) and the empty set.
    🔗
    Example 164. Finite Intersection of Open Sets.
    Any finite intersection of open sets is open.
    🔗
    Let \(U_i \subset \mathbb{R}\) be an open set for \(i=1, 2, 3, \ldots n\text{.}\) We want to prove that \(U=\displaystyle \cap_{i=1}^n U_i\) is also open.
    🔗
    If \(U\) is empty, then the result follows because the empty set is open.
    🔗
    If \(U\) is not empty, let \(x \in U\text{.}\) We must show there is an \(\epsilon > 0\) for which \(V_{\epsilon}(x) \subset U\text{.}\) By the definition of \(U\text{,}\) we know that \(x \in U_i\) for all \(i\text{.}\) Since each \(U_i\) is open, there is an \(\epsilon_i > 0\) for which \(V_{\epsilon_i}(x) \subset U_i\text{.}\) We want to find an \(\epsilon > 0\) for which \(V_{\epsilon}(x) \subset U_i\) for all \(i\text{.}\)
    🔗
    Take \(\epsilon = \min\{\epsilon_1, \epsilon_2, \epsilon_3, \ldots, \epsilon_n\}\text{.}\) Then \(V_{\epsilon}(x) \subset U_i\) for all \(i\text{.}\) Thus, \(V_{\epsilon}(x) \subset U\text{,}\) so \(U=\displaystyle \cap_{i=1}^n U_i\) is open.
    🔗
    🔗
    🔗

    Section Closed Sets

    Definition 165. Limit Point.
    A point \(x\) is a limit point of a set \(A \subset \mathbb{R}\) if every \(\epsilon\)-neighborhood \(V_{\epsilon}(x)\) intersects \(A\) at some point other than \(x\text{.}\)
    🔗
    🔗
    Theorem 166.
    A point \(x\) is a limit point of a set \(A \subset \mathbb{R}\) if and only if \(x=\lim a_n\) for some sequence \((a_n)\) contained in \(A\) with \(a_n \neq x\) for all \(n \in \mathbb{N}\text{.}\)
    🔗
    🔗
    Example 167. Every point of the closed interval \([a,b]\) is a limit point of the interval..
    Let \(x \in [a,b]\text{.}\) By the theorem, we need to show that \(x\) is the (sequential) limit of a sequence of points in the interval \([a,b]\text{.}\)
    🔗
    Suppose that \(x\) is not an endpoint of the interval, and let \(\Delta\) be the distance between \(x\) and the nearest endpoint. The the points \(x + \Delta \cdot 2^{-n}\) for \(n=1, 2, 3, \ldots\) are in \([a,b]\text{.}\) They clearly converge to \(x\text{.}\) Therefore, \(x\) is a limit point.
    🔗
    Suppose that \(x\) is an endpoint of the interval. If \(x=a\) then points \(a + (b-a) \cdot 2^{-n}\) for \(n=1, 2, 3, \ldots\) are in \([a,b]\) and converge to \(a\text{,}\) making \(a\) a limit point of \([a,b]\text{.}\) If \(x=b\) then points \(b - (b-a)\cdot 2^{-n}\) for \(n=1, 2, 3, \ldots\) are in \([a,b]\) and converge to \(b\text{,}\) making \(b\) a limit point of \([a,b]\text{.}\)
    🔗
    This shows that every point of \([a,b]\) is a limit point of \([a,b]\text{.}\)
    🔗
    🔗
    Example 168. A limit point of the closed interval \([a,b]\) must be in the interval..
    Suppose not. Suppose \(x\) is a limit point of the interval \([a,b]\) that lies outside the interval. This means that either \(x < a\) or \(x > b\text{.}\)
    🔗
    By the previous theorem, since \(x\) is a limit point of \([a,b]\) there is a sequence \((x_n) \subset [a,b]\) that converges to \(x\text{.}\) Since \((x_n) \subset [a,b]\) we know \(a \leq x_n \leq b\) for all \(n\text{.}\) By the Limit Order Theorem the limit of the sequence must satisfy \(a \leq x \leq b\text{.}\) But either \(x < a\) or \(x > b\text{.}\) This is a contradiction.
    🔗
    We conclude all limit points of \([a,b]\) must be in \([a,b]\text{.}\)
    🔗
    🔗
    NOTE: this proof can be reworked into a direct proof of the statement. Direct proofs are always better than indirect proofs. Can you make the revision?
    🔗
    🔗

    Section

    A point in \(A\) that is not a limit point of \(A\) is called an isolated point of \(A\text{.}\)
    🔗
    Definition 169. Isolated Point.
    A point \(x \in A \subset \mathbb{R}\) is an isolated point of \(A\) if it is not a limit point of \(A\text{.}\)
    🔗
    🔗
    This means you can’t get close to an isolated point of \(A\) from within \(A\text{.}\)
    🔗
    Example 170. \(x=1\) is an isolated point of \(\{1\} \cup [2,3]\text{.}\).
    Consider \(A=\{1\} \cup [2,3]\text{.}\) What are the limit points? What are the isolated points?
    🔗
    🔗
    Example 171. \(\{1\} \cup (2,3)\text{.}\).
    Consider \(A=\{1\} \cup (2,3)\text{.}\) What are the limit points? What are the isolated points?
    🔗
    🔗
    Definition 172. Closed Set.
    A set \(F \subseteq \mathbb{R}\) is called a closed set if it contains all its limit points.
    🔗
    🔗
    Canonical example, a closed and bounded interval: \([a,b]=\{x \in \mathbb{R} \, : \, a \leq c \leq b\}\text{.}\)
    🔗
    We just proved that \([a,b]\) contains all its limit points, so it is a closed set.
    🔗
    🔗

    Section Closure

    Definition 173.
    Let \(A \subseteq \mathbb{R}\text{.}\) If \(L\) is the set of limit points of \(A\text{,}\) then the closure of \(A\) is \(\overline{A} = A \cup L\text{.}\)
    🔗
    🔗
    Theorem 174.
    If \(A \subseteq \mathbb{R}\text{,}\) then \(\overline{A}\) is a closed set.
    🔗
    🔗
    The set \(\overline{A}\) is the smallest closed set that contains \(A\text{.}\)
