    Section Before break ...

    Open set of reals.
    🔗
    Definition 175. Open Set.
    A set \(U \subseteq \mathbb{R}\) is called an open set if for every \(x \in U\) there is an \(\epsilon > 0\) for which \(V_{\epsilon}(x_0) \subset U\text{.}\)
    🔗
    🔗
    Closed sets of reals, limit point of a set, and isolated point of a set.
    🔗
    Definition 176. Closed Set.
    A set \(F \subseteq \mathbb{R}\) is called a closed set if it contains all its limit points.
    🔗
    🔗
    Canonical example, a closed and bounded interval: \([a,b]=\{x \in \mathbb{R} \, : \, a \leq c \leq b\}\text{.}\)
    🔗
    Other examples of close sets: the real line \(\mathbb{R}\text{,}\) the empty set, a finite set of points.
    🔗
    🔗

    Section Closure

    Definition 177.
    Let \(A \subseteq \mathbb{R}\text{.}\) If \(L\) is the set of limit points of \(A\text{,}\) then the closure of \(A\) is \(\overline{A} = A \cup L\text{.}\)
    🔗
    🔗
    Theorem 178.
    If \(A \subseteq \mathbb{R}\text{,}\) then \(\overline{A}\) is a closed set.
    🔗
    🔗
    If this theorem is true, we will day that the set \(\overline{A}\) is the ``smallest closed set that contains \(A\text{.}\)’’
    🔗
    Proof.
    Let \(L\) denote the limit points of \(A\text{.}\) Then \(\overline{A} = A \cup L\text{.}\)
    🔗
    We must show that \(\overline{A}\) contains all its limit points. Let \(x\) be a limit point of \(\overline{A}\text{.}\) We must show \(x \in \overline{A}\text{.}\)
    🔗
    Since \(x\) is a limit point of \(\overline{A}\text{,}\) there is a sequence \((x_n) \subset \overline{A}=A \cup L\) that converges to \(x\text{.}\) We will show that \(x \in L \subset \overline{A}\text{.}\)
    🔗
    We will do so by showing there is exists a sequence (to be named later) in \(A\) that converges to \(x\text{.}\)
    🔗
    One of two things can be true of the sequence \((x_n)\text{:}\) it is either eventually in \(A\text{,}\) or it is not eventually in \(A\text{.}\) The former means that there is an \(N\) for which is \(k \geq N\) then \(x_k \in A\text{.}\) The latter means that for any \(N\) we can find a \(k \geq N\) for which \(x_k \notin A\text{.}\)
    🔗
    Consider the case where \((x_n)\) is eventually in \(A\text{.}\) This means the entire tail of \((x_n)\) is in \(A\text{,}\) so that tail is a sequence in \(A\) that converges to \(x\text{.}\) Thus \(x \in L\) by Theorem 3.2.5. of the textbook.
    🔗
    Now, consider the alternative case. If \((x_n) \subset \overline{A}\) is not eventually in \(A\text{,}\) then there is a subsequence \((x_{n_k})\) of \((x_n)\) for which \(x_{n_k} \in L \setminus A\text{.}\)
    🔗
    🔗
    🔗

    Section

    Proof.
    Define a new sequence \((y_k)\) by taking \(y_k=x_{n_k} \in L \setminus A\text{.}\) Since each \(y_k\) is a limit point of \(A\text{,}\) we can construct a sequence \((x_{k,j})_{j=1}^{\infty} \subset A\) that converges to \(y_k\text{,}\) for each \(k\text{,}\) in the following way. For \(\epsilon=1/n\text{,}\) we know that each \(V_{\epsilon}(y_k)\) contains a point of \(A\) that is different from \(y_k\text{.}\) Call this point \(x_{n,k}\text{.}\) For each \(k\text{,}\) this defines a sequence \((x_{n,k})\) that converges to \(y_k\) as \(n\) grows without bound.
    🔗
    Let \(z_k=x_{k,k} \in A\text{.}\) This defines a sequence \((z_k) \subset A\text{.}\) We claim that \((z_k) \rightarrow x\text{.}\) Consequently, \(x \in L\) by textbook theorem 3.2.5.
    🔗
    To see that \((z_k) \rightarrow x\text{,}\) let \(\epsilon > 0\) be given. Because \((x_n)\) converges to \(x\text{,}\) and because \((y_k)\) is a subsequence of \((x_n)\text{,}\) it must also converge to \(x\text{.}\) Therefore, there is an \(N_1\) for which if \(k > N_1\) then \(|y_k-x| < \epsilon/2\text{.}\) By construction, there is an \(N_2\) for which \(|z_{n} - y_k| < 1/n < \epsilon/2\) for all \(n > N_2\text{.}\) Let \(N=\max\{N_1,N_2\}\text{,}\) and let \(n > N\text{.}\) Then
    \begin{align*} |z_{n} - x| \amp = |z_{n} - y_n + y_n -x| \\ \amp < \epsilon/w + \epsilon/2 \\ \amp = \epsilon \end{align*}
    . This shows that \(x\) is a limit point of \(A\text{,}\) as desired.
    🔗
    🔗
    🔗

    Section Complements

    Open sets and closed sets have this important relationships.
    🔗
    Theorem 179.
    A set \(U \subset \mathbb{R}\) is open if and only if \(U^c\) is closed. Similarly, a set \(F \subset \mathbb{R}\) is closed if and only if \(F^c\) is open.
    🔗
    🔗
    Proof.
    Suppose \(U \subset \mathbb{R}\) is open. We musth show \(U^c\) is closed. That means we must show that \(U^c\) contains all its limit points. Let \(x\) be a limit point of \(U^c\text{.}\) Then for every \(\epsilon > 0\text{,}\) the set \(V_{\epsilon}(x)\) contains a point of \(U^c\) that is different from \(x\text{.}\) Were \(x \in U\text{,}\) then there would be an \(\epsilon > 0\) for which this was impossible! Therefore, \(x \in U^c\text{.}\)
    🔗
    Suppose \(U^c\) is closed. We will show \(U\) must be open. Let \(x \in U\text{.}\) We must find an \(\epsilon > 0\) for which \(V_{\epsilon}(x) subset U\text{.}\) Because \(U^c\) is closed, \(x\) is not a limit point of \(U^c\text{;}\) it is isolated from \(U^c\text{,}\) which means there is an \(\epsilon > 0\) for which \(V_{\epsilon}(x)\) does not intersect \(U^c\text{.}\) This means \(V_{\epsilon}(x) \subset U\text{,}\) as desired.
    🔗
    This proves the first statement.
    🔗
    To prove the second statement, note that if \(A \subset \mathbb{R}\) is a set, the \((A^c)^c=A\text{.}\) So we can prove the second statement using the first and this fact abo complements.
    🔗
    🔗
    🔗

    Section Complements

    A closing fact (haha!) about closed sets can be proved using DeMorgan’s Laws and Theorem 3.2.3, the analogous result for open sets.
    🔗
    Theorem 180.
    1. The union of a finite number of closed sets is a closed set.
      🔗
    2. The intersection of an arbitrary number of closed sets is a closed set.
      🔗
    🔗
    🔗
    Note that Theorem 3.2.3 can be summarized by saying that open sets like unions; this theorem can be summarized by saying that closed sets like intersections.
    🔗

    A set \(K \subset \mathbb{R}\) is compact if every sequences in \(K\) has a subsequence that converges in \(K\text{.}\)
