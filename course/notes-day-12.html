    Section Bolzano-Weierstrass Theorem

    Last time:
    🔗
    Theorem 122.
    Subsequences of a convergent sequence converge to the same limit as the original sequence.
    🔗
    Example 123.
    Prove that if \(0 < b < 1\) then \((b^n) \rightarrow 0\text{.}\)
    🔗
    Let \(a_n=b^n\text{.}\) Since \(0 < b < 1\) we know
    \begin{equation*} a_{n+1}=b^{n+1}=b \cdot b^n = b \cdot a_n < a_n \end{equation*}
    so we know that \((a_n)\) is monotone decreasing. Since the sequence consistes of positive numbers, it is bounded below by 0. We can invoke the Monotone Convergence Theorem to asset that \((a_n)\) converges to some real number \(\ell\text{.}\)
    🔗
    To determine the value of \(\ell\) we note that \((b^{2n})\) is a subsequence of \((a_n)\text{.}\) Therefore it also converges to \(\ell\text{.}\) But it is also true that
    \begin{align*} \lim b^{2n} \amp = \lim (b^{n}\cdot b^n) \\ \amp = \lim (b^{n}) \cdot \lim (\cdot b^n) \\ \amp = \ell \cdot \ell \end{align*}
    Therefore, \(\ell=\ell^2\text{.}\) This means \(0=\ell^2-\ell=\ell(\ell-1)\) so either \(\ell=0\) or \(\ell=1\text{.}\) Because \((a_n)\) is decreasing, its limit must be \(\ell=0\text{.}\)
    🔗
    🔗
    🔗

    Section

    Theorem 124. Bolzano-Weierstrass.
    A bounded sequence has a convergent subsequence.
    🔗
    🔗
    The idea of the proof:
    1. Since the sequence \((a_n)\) is bounded, all its terms fall within some closed bounded ionterval \(I_0=[-M,M]\)
      🔗
    2. Split this interval half; at least one half must contain an infinite number of terms of \((a_n)\) ; call that interval \(I_1\text{.}\) The length of \(I_1=\frac{1}{2} \cdot 2M\)
      🔗
    3. Pick the first term of the sequence that’s in \(I_1\) and call it \(a_{n_1}\text{.}\) (this is the first term of the subsequence that we’ll show is convergent)
      🔗
    4. Consider the tail of \((a_n)\) starting at \(N=n_1\) and repeat the previous steps: \(I_2\) is a half of \(I_1\) that contains an infinite number of terms of \((a_n)\) starting at \(M=n_1\) and call \(a_{n_2}\) the first term of the subsequence that is in \(I_2\)
      🔗
    5. This gives us a nested set of closed and bounded intervals
      \begin{equation*} I_0 \supset I_1 \supset I_2 \supset \cdots I_n \supset I_{n+1} \supset \cdots \end{equation*}
      and a subsequence \((a_{n_k})\) where \(a_{n_i} \in I_i\)
      🔗
    6. The Nested Interval Principle tells us that there is a real number \(x \in \cap_{i=0} I_i\)
      🔗
    7. we conclude by proving \((a_{n_k}) \rightarrow x\) using the fact that the lengths of the intervals \(I_i \rightarrow 0\)
      🔗
    🔗
    🔗

    Section Cauchy Sequences

    Definition 125.
    A sequence \((a_n)\) is a Cauchy sequence if for every \(\epsilon > 0\) there is an \(N in \mathbb{N}\) for which if \(n,m > N\) then \(|a_n-a_m| < \epsilon\text{.}\)
    🔗
    Note that \(|a_n-a_m| < \epsilon\) for all \(n,m > N\) means
    \begin{gather*} -\epsilon < a_n-a_m < \epsilon\\ a_m-\epsilon < a_n < a_m+\epsilon\text{,} \end{gather*}
    🔗
    All the terms of the sequence are clustering up here.
    🔗
    Theorem 126.
    Every convergent sequence is a Cauchy sequence.
    🔗
    Proof.
    Let \((x_n)\) be a convergent sequence. Call its limit \(x_0\text{.}\) We want to show that it is a Cauchy sequence. That means, we need to shows that given any \(\epsilon > 0\) there is an \(N in \mathbb{N}\) for which if \(n,m > N\) then \(|x_n-x_m| < \epsilon\text{.}\)
    🔗
    Let \(\epsilon > 0\) be given. Because we know \((x_n)\) is convergent, we know there is an \(N in \mathbb{N}\) for which if \(n > N\) then \(|x_n-x_0| < \epsilon/2\text{.}\) Take \(n,m > N\text{.}\) Then
    \begin{align*} |x_n-x_m| \amp = |x_n-x_0+x_0-x_m|\\ \amp \leq |x_n-x_0|+|x_0-x_m|\\ \amp = |x_n-x_0|+|x_m-x_0|\\ \amp < \epsilon/2 + \epsilon/2 \\ \amp = \epsilon \text{.} \end{align*}
    This shows that \((x_n)\) is a Cauchy sequence.
    🔗
    🔗
    Lemma 127.
    Every Cauchy sequence is bounded.
    🔗
    Proof.
    Let \((x_n)\) be a Cauchy sequence. Then know there is an \(N in \mathbb{N}\) for which if \(n, m > N\) then \(|x_n-x_m| < 1\text{.}\) Taking \(m=N+1\) gives \(|x_n-x_{N+1}| < 1\) so for \(n > N\text{,}\)
    \begin{gather*} -1 < x_n-x_{N+1} < 1\\ x_{N+1}-1 < x_n < x_{N+1}+1\text{,} \end{gather*}
    so \(|x_n| < |x_{N+1}| + 1\text{.}\) This give a bound for all the terms of the sequence in its tail. To accont for the possibility that one of the (finite number of) terms in the head of the sequence could be larger, we let
    \begin{equation*} M=\max\{ |x_1|, |x_2|, \ldots, |x_N|,|x_{N+1}|+1\}\text{.} \end{equation*}
    This \(M\) is a bound for the sequence.
    🔗
    🔗
    🔗

    Section Convergence is Cauchy

    Theorem 128. Cauchy Criterion.
    A sequence of real numbers converges if and only if it is a Cauchy sequence.
    🔗
    Proof.
    Let \((x_n)\) be a sequence of real numbers.
    🔗
    Suppose the sequence \((x_n)\) is convergent. We want to show \((x_n)\) is a Cauchy sequence. This was proved above.
    🔗
    Suppose the sequence \((x_n)\) is a Cauchy sequence. We want to show that \((x_n)\) converges. This means we need to find a number \(x_0\) that is the limit of \((x_n)\text{.}\)
    🔗
    By the above lemma, because \((x_n)\) is Cauchy, it is bounded. The Bolzano-Weierstrass Theorem tells us that every bonded sequence has a convergent subsequence, so our sequence \((x_n)\) has a convergent subsequence which we will call \((x_{n_k})\text{.}\) Let’s call the limit of this subsequence \(x_0\text{.}\)
    🔗
    We will show that \((x_n) \rightarrow x_0\text{.}\)
    🔗
    Let \(\epsilon > 0\) be given. Since \((x_n)\) is Cauchy, there is an \(N_1 \in \mathbb{N}\) for which if \(n,m > N_1\) we can say \(|x_n=x_m| < \epsilon/2\text{.}\) Since \((x_{n_k}) \rightarrow x_0\) there is an \(N_2 \in \mathbb{N}\) wuch that if \(n_k > N_2\) then \(|x_{n_k}-x_0| < \epsilon/2\text{.}\) Set \(N=\max\{N_1,N_2\}\text{.}\) Then if \(k > N\) it follows that \(n_k \geq k > N\) and
    \begin{align*} |x_k-x_0| \amp = |x_k-x_{n_k} + x_{n_k} -x_0|\\ \amp \leq |x_k-x_{n_k}| + |x_{n_k} -x_0|\\ \amp < \epsilon/e + \epsilon/2\\ \amp = \epsilon\text{.} \end{align*}
    This shows that \((x_n)\) is a convergent sequence with limit \(x_0\text{.}\)
    🔗
    🔗

    Section Convergence is Cauchy

    Theorem 129. Cauchy Criterion for Series.
    The series \(\sum_{k=1}^{\infty} a_k\) converges if and only if, given \(\epsilon > 0\) there exists an \(N \in \mathbb{N}\) such that whenever \(n > m \geq N\) it follows that
    \begin{equation*} |a_{m+1}+a_{m+2}+a_{m+3}+ \cdots + a_n| < \epsilon\text{.} \end{equation*}
    🔗
    Proof.
    Note that if \((s_n)\) is the sequence of \(n\)th partial sums of \(\sum_{k=1}^{\infty} a_k\text{,}\) then
    \begin{equation*} |s_n-s_m| = |a_{m+1}+a_{m+2}+a_{m+3}+ \cdots + a_n|\text{.} \end{equation*}
    and the result follows from the definition of convergense of a series.
    🔗
    🔗
    Theorem 130.
    If the series \(\sum_{k=1}^{\infty} a_k\) converges, then \((a_k) \rightarrow 0\text{.}\)
    🔗
    🔗
    Proof.
    Apply the Cauchy criterion for series using \(n=m+1\text{.}\)

    Section Topology: Open and Closed Sets

    So far in the couse we’ve talked about:
    • the nature of the real numbers (i.e., completeness),
      🔗
    • consequences of completeness of the reals, and
      🔗
    • sequences and their behavior.
      🔗
    🔗
    Now we are going to spend some time talking about different types of sets of real numbers. This will give us a vocabulary to use when talking about real valued functions of a real variable and their properties (i.e., continuity, differentiability).
    🔗
    Definition 131.
    Let \(\epsilon > 0\) be a real number. The \(\epsilon\)-neighborhood about \(x_0\) is the interval
    \begin{equation*} V_{\epsilon)(x_0) = \{ x \in \mathbb{R} \, : \, x_0-\epsilon < c < x+\epsilon\}=(x_0-\epsilon, x_0+\epsilon)\text{.} \end{equation*}
    🔗
    🔗
    Definition 132. Open Set.
    A set \(U \subseteq \mathbb{R}\) is called an open set if for every \(x \in U\) there is an \(\epsilon > 0\) for which \(V_{\epsilon)(x_0) \subset U\text{.}\)
