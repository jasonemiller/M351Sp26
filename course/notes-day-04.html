    Section Recap: The Axiom of Completeness

    We were reminded that \(\mathbb{Q}\text{,}\) though a beautiful number system (everybody say "ordered field"!), it is literally full of holes.
    🔗
    If we want geometry and algebra to be consistent ("two non-parallel lines intersect at a point") we must fill in the holes!
    🔗
    We do this using the Axiom on Completeness:
    🔗
    Theorem 30. Axiom of Completeness.
    If \(A\) is a nonempty set that is bounded above, then it has a least upper bound.
    🔗
    🔗
    With this the set of real numbers \(\mathbb{R}\) is the set \(\mathbb{Q}\) plus all the numbers we created as suprema of all possible sets that are bounded above.
    🔗
    In practice, we will continue working with the real numbers in the way we’ve learned to work with them except that we’re learning more about the reals. For example, they form a complete ordered field!
    🔗
    🔗

    Section Working with Suprema (and Infima)

    Definition 31.
    Let \(A \subseteq \mathbb{R}\) be nonempty. The real number \(s\) is a least upper bound (aka supremum) of \(A\) if
    1. \(s\) is an upper bound for \(A\text{,}\) and
      🔗
    2. if \(M\) is another upper bound, then \(s \leq M\text{.}\)
      🔗
    The real number \(\ell\) is a greatest lower bound (aka infimum) of \(A\) if
    1. \(\ell\) is an lower bound for \(A\text{,}\) and
      🔗
    2. if \(m\) is another lower bound, then \(m \leq \ell\text{.}\)
      🔗
    🔗
    Theorem 32.
    Let \(A \subseteq \mathbb{R}\) be a nonempty set that is bounded above. Let \(s \in \mathbb{R}\) be an upper bound for \(A\text{.}\) Then \(s=\sup(A)\) if and only if, for every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon < a\text{.}\)
    🔗
    🔗
    🔗

    Section Working with Suprema (and Infima)

    Theorem 33.
    Let \(A \subseteq \mathbb{R}\) be a nonempty set that is bonded above. Let \(s \in \mathbb{R}\) be an uppoer bound for \(A\text{.}\) Then \(s=\sup(A)\) if and only if, for every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon < a\text{.}\)
    🔗
    🔗
    Proof.
    (\(\Longrightarrow\text{,}\) Contradiction)
    🔗
    Assume \(s=\sup(A)\text{.}\) We must show that for every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon < a\text{.}\)
    🔗
    Suppose not. Suppose the opposite of what we want to conclude is true. The negation of ``or every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon < a\)’’ is ``there is an\(\epsilon_0 > 0\) where \(s-\epsilon_0 \geq a\) for all \(a \in A\text{.}\)’’
    🔗
    This assumption implies that \(s-\epsilon_0\) is an upper bond for \(A\text{.}\) Since \(s=\sup(A)\) is the least upper bound for \(A\text{,}\) it follows that \(s < s-\epsilon_0\text{.}\) But this is impossible.
    🔗
    Therefore, our assumption that there is an\(\epsilon_0 > 0\) where \(s-\epsilon_0 \geq a\) for all \(a \in A\) must be false. It’s opposite must be true. Therefore, given any \(\epsilon > 0\text{,}\) there exists an \(a \in A\) for which \(s-\epsilon < a\text{.}\) This completes the proof of this part.
    🔗
    (\(\Longleftarrow\text{,}\) Contradiction)
    🔗
    Let \(s \in \mathbb{R}\) be an upper bound for \(A\text{.}\) Also assume that for every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon < a\text{.}\) We must show that \(s=\sup(A)\text{.}\) By definition, this means we must show that (1) \(s\) is an upper bound for \(A\text{,}\) and (2) if \(M\) is any other upper bound for \(A\) then \(s \leq M\text{.}\)
    🔗
    Since we are already assuming the former, we need only prove the latter.
    🔗
    Let \(M\) be another upper bound for \(A\text{.}\)
    🔗
    Suppose \(M < s\text{.}\) Choose \(\epsilon_0 = s-M\) so that \(M=s-\epsilon_0\text{.}\)
    🔗
    Since \(s\) has the property that, for every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon < a\text{,}\) we know there is an \(a_0 \in A\) for which \(M=e-\epsilon_0 < a_0\text{.}\)
    🔗
    But this means that \(M\) is not an upper bound for \(A\text{!}\) Contradiction.
    🔗
    Consequently, our supposition that \(M < s\) was incorrect. We conclude that \(M \grq s\text{.}\) With this, we have shows that \(s\) satisfies the definition of what it means to be the supremum of \(A\text{.}\) QED.
    🔗
    🔗
    🔗

    Section More on Suprema

    Example 34. Supremum of a subset.
    If \(A_1 \subseteq A_2\) are sets of real numbers that are both bounded above, then \(\sup(A_1) \leq \sup(A_2)\text{.}\)
    🔗
    🔗
    🔗
    Example 35. Supremum of a union.
    If \(A_1\) and \(A_2\) are sets of real numbers that are bounded above, then
    \begin{equation*} \sup(A_1 \cup A_2) = \max\{\sup(A_1), \sup(A_2)\}\text{.} \end{equation*}
    🔗
    🔗
    🔗
    Can you come up with a relationship between the suprema of two sets and the suprema of their intersection? Their infima?
    🔗
    🔗

    Section Consequences of Completeness

    Well, you can’t to this with the rationals!
    🔗
    Theorem 36. Nested Interval Property.
    For each \(n \in \mathbb{N}\text{,}\) define \(I_n=[a_n,b_n] \subseteq \mathbb{R}\) so that \(I_n \subseteq I_{n-1}\text{.}\) Then \(\bigcap_{n=1}^{\infty} I_n \neq \emptyset\text{.}\)
    🔗
    The intervals in this theorem are all closed and bounded intervals of reals that form a sequence of nested intervals.
    \begin{equation*} I_1 \supseteq I_2 \supseteq I_3 \supseteq \cdots \supseteq I_n \supseteq \cdots \end{equation*}
    🔗
    Proof.
    Before we begin, notice how the endpoints of the intervals \(I_k\) are related to one another.
    🔗
    Since \(I_1 \subseteq I_k\) for all \(k\text{,}\) we know \(a_k \leq b_1\) for all \(k \in \mathbb{N}\text{.}\) This shows that the set \(A=\{a_n \, : \, n \in \mathbb{N} \}\) is bounded above. Let \(x=\sup(A)\text{.}\)
    🔗
    Consider some \(I_n\text{.}\) We claim that \(x \in I_n\text{.}\)
    🔗
    Since \(x\) is an upper bound for \(A\text{,}\) we know \(a_n \leq x\) for all \(n \in \mathbb{N}\text{.}\)
    🔗
    Since \(b_n\) is an upper bound for \(A\text{,}\) and since \(x\) is the least upper bound for \(A\) we can conclude that \(x \leq b_n\text{.}\) This shows \(x \in I_n\text{.}\)
    🔗
    Since \(n\) above was arbitrary, we have shown that \(x \in I_n\) for all \(n \in \mathbb{N}\text{.}\)
    🔗
    Consequently, \(x \in \bigcap_{n=1}^{\infty}I_n\text{,}\) as desired.
    🔗
    🔗

    Theorem 38. Archimedean Properties.
    1. Given any \(x \in \mathbb{R}\text{,}\) there is an \(n \in \mathbb{N}\) with \(n > x\text{.}\)

      🔗
    2. Given any \(x \in \mathbb{R}\) with \(x>0\text{,}\) there is an \(n \in \mathbb{N}\) with \(\frac{1}{n} < x\text{.}\)
