    Section Continuity and Uniform Continuity

    We’ve built on our understanding of functional limits to establish the concept of continuity and some of its consequences:
    1. What it means for a function to be continuous on a set.
      🔗
      🔗
    2. What is means for a function to be uniformly continuous on a set.
      🔗
      🔗
    3. The algebraic combination of continuous functions and the composition of continuous functions are also continuous.
      🔗
      🔗
    4. The image of a compact set under a continuous function is also compact.
      🔗
      🔗
    5. A continuous function on a compact domain attains both an absolute maximum value and an absolute minimum value.
      🔗
      🔗
    6. A function that is continuous at every point of a compact set is uniformly continuous on that compact set.
      🔗
      🔗
    🔗
    I also mentioned without proof the following consequences of continuity:
    1. If \(E \subset \mathbb{R}\) is a connected set, and if \(f:E \rightarrow \mathbb{R}\) is continuous on \(E\) then \(f(E)\) is a connected set.
      🔗
      🔗
    2. If \(f:\mathbb{R} \rightarrow \mathbb{R}\) is continuous on \(\mathbb{R}\text{,}\) and if \(B \subseteq \mathbb{R}\) is an open set, then \(f^{-1}(B)\) is an open set.
      🔗
      🔗
    🔗
    The first of these gives us ...
    🔗
    Theorem 222. Intermediate Value Theorem.
    If \(f:[a,b] \rightarrow \mathbb{R}\) is continuous on \([a,b]\text{,}\) and if \(L\) is any number between \(f(a)\) and \(f(b)\text{,}\) then there is a \(c \in [a,b]\) at which \(f(c)=L\text{.}\)
    🔗
    🔗
    This is a very important consequence of the completeness of the reals.
    🔗
    🔗

    Section Differentiability

    In Calculus 1, we learn about the property of differentiability, what it says about functions, and how it can be used to learn other things about functions and to model real-world phenomena.
    🔗
    Definition 223.
    Let \(g : A \rightarrow \mathbb{R}\) be a function on an interval \(A \subseteq \mathbb{R}\text{.}\) Given \(c \in A\text{,}\) the derivative of \(g\) at \(c\) is defined by
    \begin{align*} g'(c) \amp = \lim_{x \rightarrow c} \frac{g(x)-x(c)}{x-c} \amp \end{align*}
    provided the limit exists. In this case we say \(g\) is differentiable at \(x=c\). If \(g'\) exists for all points \(c \in A\text{,}\) we say that \(g\) is differentiable on \(A\).
    🔗
    🔗
    🔗

    Theorem 224. Differentiablity implies Continuity.

    If \(f(x)\) is differentiable at \(x=c\text{,}\) then \(f\) is continuous at \(x=c\text{.}\)
    🔗
    🔗
    This means that continuity is prerequisite for differentiability. (The contrapositive of the above.)
    🔗

    Proof.

    Suppose that \(f\) is differentiable at \(x=c\text{.}\) This means that the following limit exists and is finite:
    \begin{align*} \lim_{x \rightarrow c} \frac{f(x)-f(c)}{x-c} \amp = f'(c) \end{align*}
