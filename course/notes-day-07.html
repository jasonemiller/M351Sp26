    Section Convergence of Sequences

    Definition 76.
    A sequence \((x_n)\) converges to a real number \(x_0\)if for every \(\epsilon >0\) there is an \(N=N(\epsilon) \in \mathbb{N}\) with the property that if \(n >N\) then \(|x_n-x_0| < \epsilon\text{.}\)
    🔗
    Fact 77.
    "As \(n\) gets bigger and bigger, \(x_n\) gets closer and closer to \(x_0\text{.}\)"
    🔗
    🔗
    Last time, we proved:
    1. \(\displaystyle \displaystyle \lim_{n \rightarrow \infty} \frac{1}{n}=0\)
      🔗
    2. \(\displaystyle (started) \displaystyle \lim_{n \rightarrow \infty} \frac{3n+1}{7n-4}=\frac{3}{7}\)
      🔗
    🔗
    Example 78. Convergent sequence: powers of 2.
    Prove that \(\displaystyle \lim_{n \rightarrow \infty} \frac{1}{2^n}=0\text{.}\)
    🔗
    SCRATCH WORK:
    \begin{align*} \left|\frac{1}{2^n}-0\right| \amp = \left|\frac{1}{2^n}\right|\\ \amp = \frac{1}{2^n} \text{ because values are all positive }\text{.} \end{align*}
    We want to make this arbitrarily small. That is, we want to show that for large \(n\text{,}\) this can be made smaller than \(\epsilon\text{.}\) How big does \(n\) need to be? We work to find out. Specifically, we want to solve the inequality
    \begin{equation*} \frac{1}{2^n} < \epsilon\text{.} \end{equation*}
    🔗
    Steps to solve for \(n\) follow:
    \begin{gather*} \frac{1}{2^n} < \epsilon\\ 2^n > \frac{1}{\epsilon}\\ \log_2(2^n) > \log_2(\frac{1}{\epsilon})\\ n\log_2(2) > -\log_2(\epsilon)\\ n > -\log_2(\epsilon)\text{.} \end{gather*}
    On the left side of the last inequality is the candidate for the \(N\) we seek, except that it isn’t a natural number. So we set \(N=\lceil -\log_2(\epsilon) \rceil\text{.}\)
    🔗
    We are now ready to prove the limit statement for the sequence \((2^{-n})\text{.}\)
    🔗
    Note 79.
    Lots of proofs of convergence dispense with the requirement that our \(N\) be a natural number; the important aspect of \(N\) is that is ia a lower bound for the choices of \(n\) that make our expression \(|x_n-x_0|\)less than \(\epsilon\text{.}\)
    🔗
    🔗
    Proof.
    Let \(\epsilon > 0\) be given. Choose \(N=\lceil -\log_2(\epsilon) \rceil\) and let \(n > N\text{.}\) For this choice of \(n\) we see that
    \begin{align*} \left|\frac{1}{2^n}-0\right| \amp = \left|\frac{1}{2^n}\right|\\ \amp = \frac{1}{2^n}\\ \amp < \frac{1}{2^N} \text{ since }n> N\\ \amp < \frac{1}{2^{-\log_2(\epsilon)}} \text{ since } N > -\log_2(\epsilon)\\ \amp < \frac{1}{\epsilon^{-1}}\\ \amp = \epsilon \end{align*}
    This shows that for \(n>N\text{,}\) we have \(\left|\frac{1}{2^n}-0\right| < \epsilon\text{,}\) as desired.
    🔗
    🔗
    🔗

    Section Divergence of Sequences

    A sequence that does not converge is said to diverge.
    🔗
    Definition 82.
    A sequence \((x_n)\) diverges if it does not converge to a real number \(x_0\text{.}\) Negating the definition of convergence, this means that there exists an \(\epsilon>0\) for which, regardless of your choice of \(N \in \mathbb{N}\) one can find an \(n_0 > N\) for which \(|x_n-x_0|> \epsilon\text{.}\)
    🔗
    Example 83. Divergent sequence.
    Define the sequeqnce \((x_n)\) by \(x_n=n\text{.}\) Prove that this sequence diverges.
    🔗
    🔗
    Example 84. Divergent sequence.
    Define the sequeqnce \((x_n)\) by \(x_n=(-1)^n\text{.}\) Prove that this sequence diverges.
