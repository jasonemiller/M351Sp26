    Section Consequences of Completeness

    Theorem 41. Density of \(\mathbb{Q}\) in \(\mathbb{R}\).
    Let \(a, b, \in \mathbb{R}\) with \(a \neq b\text{.}\) Then there is an \(r \in \mathbb{Q}\) with \(a < r < b\text{.}\)
    🔗
    Proof.
    Let \(a\) and \(b\) be real numbers with \(a < b\text{.}\) We need to find integers \(m\) and \(n\) where
    \begin{equation*} a < \frac{m}{n} < b\text{.} \end{equation*}
    🔗
    First, we look for the \(n\text{.}\) It’s the denominator of our rational number, so the closer \(x\) and \(y\) are to one another, the larger \(n\) will be. Consider the real number \(b-a > 0\text{.}\)
    🔗
    By the Archimedean Property of real numbers, there is an \(n\) with \(0 < \frac{1}{n} < b-a\text{.}\) Now let’s start looking for \(m\text{.}\)
    🔗
    We want
    \begin{equation*} a < \frac{m}{n} < b\text{.} \end{equation*}
    This is equivalent to wanting
    \begin{equation*} na < m < nb\text{.} \end{equation*}
    We choose the smallest integer \(m\) for which
    \begin{equation*} m-1 < na < m\text{.} \end{equation*}
    (This is \(na\) rounded up to the nearest integer.). It follows that
    \begin{equation*} a < \frac{m}{n}\text{.} \end{equation*}
    🔗
    Now we want to relate our rational number to \(b\text{.}\) Since \(m-1 < na\) and \(\frac{1}{n} < b-a\text{,}\) we can write
    \begin{align*} m \amp \leq na+1\\ \amp < n(b-\frac{1}{n})+1\\ \amp = nb-1+1\\ \amp = nb\text{.} \end{align*}
    Consequently, \(\frac{m}{n} <b\text{.}\)
    🔗
    We conclude that \(a < \frac{m}{n} < b\text{,}\) as desired.
    🔗

    A set \(A\) is finite if there is an \(n \in \mathbb{N}\) for which the elements of the set \(\{1,2,3,4,\ldots n\}\) and the elements of the set \(A\) can be put in 1-1 correspondence.

    Definition 44. Function.
    A function from a set \(A\) to a set \(B\) is a rule \(f\) that assigns to each input \(a \in A\) exactly one output \(f(a)=b \in B\text{.}\)
    🔗
    🔗
    Definition 45. one-to-one function.
    A function \(f: A \rightarrow B\) is one-to-one if \(f(x)=f(y)\) means that \(x=y\text{.}\)
    🔗
    🔗
    What are some examples of one-to-one functions?
    🔗
    Fact 46.
    A function \(f\) that is one-to-one has an inverse \(f^{-1}:B \rightarrow A\text{.}\) Inverse functions are characterized by \((f^{-1} \circ f)(a)=a\) for all \(a \in A\text{,}\) and \((f \circ f^{-1})(b)=b\) for all \(b \in f(A) \subseteq B\text{.}\)
    🔗
    🔗
    Definition 47. onto function.
    A function \(f: A \rightarrow B\) is onto if for every \(b \in B\) there is an \(a \in A\) with \(f(a)=b\text{.}\)

    Definition 48. bijection.
    A function \(f: A \rightarrow B\) is a bijection if it is both one-to-one and onto.
    🔗
    🔗
    The cardinality of a set \(A\) is the number of elements in \(A\text{.}\)
    🔗
    If a set is finite, the cardinality is the number of elements in the set. If a set is not finite, we say it is an infinite set.
    🔗
    Definition 49.
    Two sets \(A\) and \(B\) have the same cardinality if there is a bijection \(\phi:A \rightarrow B\text{.}\)
    🔗
    🔗

    Section Infinite Sets

    Theorem 54.
    The set of rational numbers is countable.
    🔗
    🔗
    Proof.
    Let \(\mathbb{Q}_{-}\) be all the negative rationals and let \(\mathbb{Q}_{+}\) be all the positive rationals. We will show that \(\mathbb{Q}_{+}\) is countable.
    🔗
    Once we see that \(\mathbb{Q}_{+}\) is countable, we will have an enumeration of its elements \(\{q_1,q_2,q_3,\ldots\}\text{.}\) It follows that \(\mathbb{Q}_{-}\) is countable and has some enumeration \(\{p_1,p_2,p_3,\ldots \}\text{.}\) This allows us to arrange the elements of \(\mathbb{Q} = \mathbb{Q}_{-} \cup \{0\} \cup \mathbb{Q}_{+}\) into an array with two rows:
    \(0\) \(q_1\) \(q_2\) \(q_3\) \(\cdots\)
    \(p_1\) \(p_2\) \(p_3\) \(p_4\) \(\cdots\)
    🔗
    We can enumerate this array by snaking through the array in a way that defines a bijection from \(\mathbb{N}\) to the array.
    🔗
    This demonstrates that \(\mathbb{Q}\) is countable, as claimed.
    🔗
    To see that \(\mathbb{Q}_{+}\) is countable, arrange the rational numbers in a doubly-infinite array as follows:
    \(1/1\) \(2/1\) \(3/1\) \(4/1\) \(\cdots\)
    \(1/2\) \(2/2\) \(3/2\) \(4/2\) \(\cdots\)
    \(1/3\) \(2/3\) \(3/3\) \(4/3\) \(\cdots\)
    \(1/4\) \(2/4\) \(3/4\) \(4/4\) \(\cdots\)
    \(\vdots\) \(\vdots\) \(\vdots\) \(\vdots\) \(\vdots\)
    . All rational numbers show up multiple times in this array. For each rational number in the array, we throw away all copies of the rational number except the copy that is in lowest terms. When we throw away a rational number, we shift all the entries in its row to the left to fill the gap it leaves (and the gaps they create when they shift).
    🔗
    This leaves us an array of rational numbers in lowest terms.
    🔗
    By snaking through the array starting in the upper left corner and moving anti-diagonally (transverse to the array’s diagonal entries), we create an enumeration from \(\mathbb{N}\) to \(\mathbb{Q}_{+}\text{.}\) This shows that \(\mathbb{Q}_{+}\) is countable.
    🔗
    This concludes the proof that \(\mathbb{Q}\) is countable.
    🔗
    🔗
    🔗

    Section Infinite Sets

    Theorem 55.
    The set of real numbers is uncountable.
    🔗
    🔗
    Proof.
    (Contradiction). Suppose \(\mathbb{R}\) is countable. Then there exists an enumeratioon \(\phi:\mathbb{N} \rightarrow \mathbb{R}\text{.}\) Let \(x_n=\phi(n)\) so that
    \begin{equation*} \mathbb{R} = \{ x_1,x_2,x_3,x_4,\ldots\}\text{.} \end{equation*}
    We are going to construct a nested sequence of closed and bounded intervals that and use the Nested Interval Principle to generate a contradiction.
    🔗
    Let \(I_1 \subseteq \mathbb{R}\) be a closed and bounded interval that does not contain \(x_1\text{.}\)
    🔗
    Let \(I_2 \subseteq I_1\) be a closed and bounded interval that does not contain \(x_2\text{.}\) Why can we do this? If \(x_2 \notin I_1\text{,}\) then we can take \(I_2=I_1\text{.}\) If \(x_2 \in I_1\text{,}\) then \(x_2\) divides \(I_1\) into two complementary closed intervals \(A\) and \(B\) with \(I_2=A \cup B\) and \(A \cap B = \{x_2\}\text{.}\) Take \(I_2 \subset A\) so that \(a_x \notin I_2\text{.}\)
    🔗
    Continue in this way to define closed and bounded intervals \(I_n\) with \(I_n \subseteq I_{n-1}\) and \(x_n \notin I_n\text{.}\)
    🔗
    Consider \(I = \bigcap_{n=1}^{\infty} I_n\text{.}\) The Nested Interval Property tells us that \(I \subset \mathbb{R}\) is nonempty. Let \(x_0 \in I\text{.}\) The Nested Interval Property tells us that \(x_0\) is a real number.
    🔗
    Since \(\mathbb{R}\) is countable, this \(x_0\) is in our enumeration, so \(x_0=x_N\) for some \(N \in \mathbb{N}\text{.}\) But \(x_N \notin I_N\text{,}\) so \(x_=x_0 \notin I\text{.}\) This is a contradiction.
    🔗
    Therefore, our supposition that \(\mathbb{R}\) is countably infinte was false. The opposite is true: \(\mathbb{R}\) is uncountable.
    🔗
    🔗
    Corollary 56.
    The set of irrational numbers is uncountable.
