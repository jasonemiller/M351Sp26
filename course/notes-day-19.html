    Section

    Example 211. \(f(x)=7x+5\) is continuous at \(x=2\).
    Let \(\epsilon > 0\) be given. According to Definition 209 we need to show there is a \(\delta > 0\) for which \(|x-2| < \delta\) implies \(|f(x)-f(2)|=|(7x+5)-(7 \cdot 2+5)| < \epsilon\text{.}\)
    🔗
    Choose \(\delta = \min\{1, \epsilon/7\}\) and consider \(x \in \mathbb{R}\) for which \(|x-2| < \delta\text{.}\) Then we can say
    \begin{align*} |(7x+5)-(7 \cdot 2+5)| \amp = |7x+5-19| \\ \amp = |7x-14| \\ \amp = |7(x-2)| \\ \amp = 7 \cdot |x-2|\\ \amp < 7 \cdot \epsilon/7 \text{ because } |x-2| < \epsilon/7\\ \amp = \epsilon \end{align*}
    This shows \(f(x)=7x+5\) is continuous at \(x=2\text{.}\)
    🔗
    🔗
    Example 212. \(f(x)=7x+5\) is continuous on \(\mathbb{R}\).
    Let \(c \in \mathbb{R}\text{,}\) and let \(\epsilon > 0\) be given. According to Definition 209 we need to show there is a \(\delta > 0\) for which \(|x-c| < \delta\) implies \(|f(x)-f(c)|=|(7x+5)-(7 \cdot c+5)| < \epsilon\text{.}\)
    🔗
    Choose \(\delta = \epsilon/7\) and consider \(x \in \mathbb{R}\) for which \(|x-c| < \delta\text{.}\) Then we can say
    \begin{align*} |(7x+5)-(7 \cdot c+5)| \amp = |7x+5-7c-5| \\ \amp = |7x-7c| \\ \amp = |7(x-c)| \\ \amp = 7 \cdot |x-c|\\ \amp < 7 \cdot \epsilon/7 \text{ because } |x-c| < \epsilon/7\\ \amp = \epsilon \end{align*}
    This shows \(f(x)=7x+5\) is continuous at \(x=c\text{.}\) Since the choice of \(x=c\) was an arbitrary real number, the above shows that \(f(x)=7x+5\) is continuous at all reals.
    🔗
    🔗
    🔗

    Example 213. \(f(x)=x^2\) is continuous at \(x=5\).
    Let \(\epsilon > 0\) be given. According to Definition 209 we need to show there is a \(\delta > 0\) for which \(|x-5| < \delta\) implies \(|f(x)-f(2)|=|x^2-25| < \epsilon\text{.}\)
    🔗
    Choose \(\delta = \min\{1, \epsilon/11\}\) and consider \(x \in \mathbb{R}\) for which \(|x-5| < \delta\text{.}\) Then because \(|x-5| < \delta \leq 1\) we know \(-1 < x-5 < 1\) and \(9 < x+5 < 11\) so \(|x+5| < 11\text{.}\) We can also say
    \begin{align*} |x^2-25| \amp = |x-5| \cdot |x+5| \\ \amp < |x-5| \cdot 11 \text{ because } x-5| < 1\\ \amp < \epsilon/11 \cdot 11 \text{ because } x-5| < \epsilon/11\\ \amp = \epsilon \end{align*}
    This shows \(f(x)=x^2\) is continuous at \(x=5\text{.}\)

    Section The Squaring Function is Continuous

    Example 214. \(f(x)=x^2\) is continuous on \(\mathbb{R}\).
    We need to show \(f(x)\) is continuous at every real number. Let \(c \in \mathbb{R}\text{.}\) We will show that \(f(x)\) is continuous at \(x=c\text{.}\)
    🔗
    Let \(\epsilon > 0\) be given. According to Definition 209 we need to show there is a \(\delta > 0\) for which \(|x-c| < \delta\) implies \(|f(x)-f(c)|=|x^2-c^2| < \epsilon\text{.}\)
    🔗
    Choose \(\delta = \min\{1, \epsilon/(1+2|c|)\}\) and consider \(x \in \mathbb{R}\) for which \(|x-c| < \delta\text{.}\) Then because \(|x-c| < \delta \leq 1\) we know \(-1 < x-c < 1\) and \(2c-1 < x+c < 2c+1\) so \(|x+c| < \max\{|2c \pm 1|\}\text{.}\)
    🔗
    Notice that if \(c \geq 0\) then \(|2c-1| < |2c+1|=2c+1=2|c|+1\text{.}\) On the other hand, if \(c \leq 0\)m then \(|2c-1| \geq |2c+1|\) and \(|2c-1|=|2(-|c|)-1|=|2\cdot|c|+1|=2|c|+1\) so \(|2c+1| \leq 2|c|+1\text{.}\) Therefore, regardless of the sign of \(c \in \mathbb{R}\) we can say \(\max\{|2c \pm 1|\} \leq 2|c|+1\text{.}\) It follows that if \(|x-c| < 1\) then \(|x+c| < 2|c|+1\text{.}\)
    🔗
    With this established, choose \(x \in \mathbb{R}\) so that \(|x-c| < \delta\text{.}\) Then
    \begin{align*} |x^2-c^2| \amp = |x-c| \cdot |x+c| \\ \amp < |x-c| \cdot (2|c|+1) \text{ because } |x-5| < 1\\ \amp < \epsilon/(2|c|+1) \cdot (2|c|+1) \text{ because } |x-c| < \epsilon/(2|c|+1)\\ \amp = \epsilon \end{align*}
    This shows \(f(x)=x^2\) is continuous at \(x=c\text{.}\) Since \(c\) was arbitrary, this shows that \(f(x)=x^2\) is continuous on all \(\mathbb{R}\text{.}\)

    Theorem 215. Alternate Characterization of Continuity.
    Let \(f:A \rightarrow \mathbb{R}\) and let \(c \in A\text{.}\) The function \(f\) is continuous at \(c \in A\) if and only if one of (Item 1), (Item 2), or (Item 3) below holds:
    1. for all \(\epsilon > 0\) there is a \(\delta > 0\) such that \(|x-c| < \delta\) (and \(x \in A\)) implies \(|f(x)-f(c)| < \epsilon\text{;}\)
      🔗
      🔗
    2. For all \(\epsilon\)-neighborhoods \(V_{\epsilon}(f(c))\) of \(f(c)\) there exists a \(\delta\)-neighborhood \(V_{\delta}(c)\) of \(c\) with the property that for \(x \in V_{\delta}(c) \cap A\) we have \(f(x) \in V_{\epsilon}(f(c))\text{;}\) and,
      🔗
      🔗
    3. For all \((x_n) \subset A\) with \((x_n)\) converging to \(c \in A\) it follows that \((f(x_n))\) converges to \(f(c)\text{.}\)
      🔗
      🔗
    4. If \(c\) is a limit point of \(A\text{,}\) the previous three statement are equivalent to \(\displaystyle \lim_{x \rightarrow c} f(x)=f(c)\text{.}\)
      🔗
      🔗
    🔗
    🔗
    Proof.
    ( (Item 1) if and only if (Item 2) )
    🔗
    Statement (Item 1) is the definition of continuity at the point \(x \in A\text{,}\) and (Item 2) is its translation into topological terms. This establishes the equivalence.
    🔗
    ( (Item 1) if and only if (Item 3) )
    🔗
    We’ll prove (Item 3) implies (Item 1) by contradiction. Assume (Item 3), and also assume that (Item 1) is not true. That is, assume there is an \(\epsilon_0 > 0\) such that for all \(\delta > 0\) there is an \(x \in A\) with \(|x-c| < \delta\) where \(|f(x)-f(c)| > \epsilon_0\text{.}\)
    🔗
    Consider \(\delta_n=1/n\text{.}\) Fix \(n\text{.}\) For that \(n\text{,}\) there is a point in \(V_{\delta_n}(c) \cap A\text{,}\) call the point \(x_n\text{,}\) at which \(|f(x_n)-f(c)| > epsilon_0\text{.}\) For each \(n\text{,}\) find the analogous point \(x_n\) with this property and we get a sequence \((x_n) \subset A\) that converges to \(c\text{.}\)
    🔗
    Notice that this sequence was constructed so it has the property that the sequence of values \((f(x_n))\) does not converge to \(f(c)\text{.}\) This contradicts (Item 3).
    🔗
    If follows that (Item 1) must be true. Hence (Item 3) implies (Item 1).
    🔗
    We prove the reverse implication directly.
    🔗
    Let \((x_n) \subset A\) be a sequence that converges to \(c\text{.}\) We must show that the sequence \((f(x_n))\) converges to \(f(c)\text{.}\) Let \(\epsilon > 0\) be given.
    🔗
    By (Item 1), we know there is a \(\delta > 0\) such that if \(|x-c| < \delta\) and \(x \in A\) then \(|f(x)-f(c)| < \epsilon\text{.}\) Since \((x_n)\) converges to \(c\) we know there is an \(N=N(\epsilon)\) such that if \(n > N\text{,}\) then \(|x_n-c| < \delta\text{.}\) For the same \(n\text{,}\) it follows that \(|f(x_n)-f(c)| < \epsilon\text{.}\)
    🔗
    This proves (Item 3) holds and establishes Item 1 if and only if Item 3.
    🔗
    ( (Item 1) if and only if (Item 4) )
    🔗
    This can be established by noting that the difference between (Item 1) and the functional limit of (Item 4) is the case \(x=c \in V_{\delta}(c) \cap A\text{.}\) When \(x=c\text{,}\) we get \(f(x)=f(c)\) so that \(f(c) \in V_{\epsilon}(f(c))\text{.}\)
