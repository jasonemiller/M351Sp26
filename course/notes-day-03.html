    Section The Axiom of Completeness

    What is the author trying to do? No, let me ask a more general question. What are some vocab terms that have come up, in class and the reading?
    🔗
    • DeMorgan’s Law
      🔗
    • supremum, infimum
      🔗
    • set bounds: upper, lower, total
      🔗
    • triangle inequality
      🔗
    • (proof techniques)
      🔗
    • \(\mathbb{N}\text{,}\) \(\mathbb{Z}\text{,}\) \(\mathbb{Q}\text{,}\) \(\mathbb{Q}^c\text{,}\) \(\mathbb{R}\text{,}\)
      🔗
    🔗
    What are the basic operations endowed to \(\mathbb{N}\text{,}\) \(\mathbb{Z}\text{,}\) and \(\mathbb{Q}\text{?}\)
    🔗
    Note how these n umber systems are nested and how one allows us to get to the next until we get to \(\mathbb{Q}\text{.}\)
    🔗
    To get to \(\mathbb{Z}\) from \(\mathbb{N}\) we add the additive inverses to \(\mathbb{N}\) and zero.
    🔗
    To get to \(\mathbb{Q}\) from \(\mathbb{Z}\) we add the multiplicative inverses to \(\mathbb{Z}\text{.}\)
    🔗
    It’s not clear how to get from \(\mathbb{Q}\) to \(\mathbb{R}\) ... it’s like we can’t do it without working from the outside of \(\mathbb{Q}\text{.}\) This is where the Axiom of Completeness comes into play; it is how we pass from \(\mathbb{Q}\) to \(\mathbb{R}\text{.}\)
    🔗
    Theorem 22. Axiom of Completeness.
    If \(A\) is a nonempty set that is bonded above, then it has a least upper bound.
    🔗
    🔗
    The set \(\mathbb{Q}\) does not obey the Axiom of Completeness because \(\mathbb{Q}\) doesn’t have sufficient numbers (i.e. the irrationals). To get the reals, we take \(\mathbb{Q}\) and add all the numbers required for the new set to satisfy the Axiom of Completeness.
    🔗
    This addition completes the rationals and gives us the real number system, \(\mathbb{R}\text{.}\)
    🔗
    Our take-away from this process should be that the reals are a complete number system that have all the nive field properties of the rationals.
    🔗
    🔗

    Section Maxima and suprema

    Definition 23.
    A real number \(a_0\) is a maximum of \(A \subseteq \mathbb{R}\) if
    1. \(a_0 \in A\text{,}\) and
      🔗
    2. \(a \leq a_0\) for all \(a \in A\text{.}\)
      🔗
    🔗
    A real number \(a_0\) is a minimum of \(A \subseteq \mathbb{R}\) if
    1. \(a_0 \in A\text{,}\) and
      🔗
    2. \(a \geq a_0\) for all \(a \in A\text{.}\)
      🔗
    🔗
    🔗
    Example 24.
    A finite set of real numbers always has both a minimum and a maximum.
    Answer.
    🔗
    🔗
    Example 25.
    The open interval \((0,1)=\{x \in \mathbb{R} \, : \, 0 < x < 1 \}\text{.}\) What is its maximum? What is its minimum?
    Answer.
    This set does not have a maximum. One might want to say 1 is a maximum, but it is not an element of the set, so it cannot be a maximum. If you take a number that is a tiny bit smaller, then you can find a number in the interval that is bigger than your new number, so your new number can’t be a maximum.
    🔗
    The same type of argument explains why the interval does not have a minimum, either.
    🔗
    🔗
    🔗
    🔗

    Section Maxima and suprema

    Definition 26.
    Let \(A \subseteq \mathbb{R}\) be nonempty. The real number \(s\) is a least upper bound (aka supremum) of \(A\) if
    1. \(s\) is an upper bound for \(A\text{,}\) and
      🔗
    2. if \(M\) is another upper bound, then \(s \leq M\text{.}\)
      🔗
    The real number \(\ell\) is a greatest lower bound (aka infimum) of \(A\) if
    1. \(\ell\) is an lower bound for \(A\text{,}\) and
      🔗
    2. if \(m\) is another lower bound, then \(m \leq \ell\text{.}\)
      🔗
    🔗
    Example 27.
    For each of the following sets, determine its infimum, supremum, minimum, and maximum or decide that it does not exist.
    1. \(\displaystyle A=\{ \frac{1}{n} \, : \, n \in \mathbb{N} \}\)
      🔗
    2. \(\displaystyle B=\{ \frac{1}{n} \, : \, n \in \mathbb{Z} \setminus \{0\} \, \}\)
      🔗
    3. \(\displaystyle C=[0,1] \cup [2,3]\)
      🔗
    4. \(\displaystyle D=\bigcap_{i=1}^{\infty} [-\frac{1}{n},1+\frac{1}{n}]\)
      🔗
    5. \(\displaystyle E=\{ x \in \mathbb{R} \, : \, x < 0\}\)
      🔗
    6. \(\displaystyle F=\bigcap_{i=1}^{\infty} (1-\frac{1}{n},1+\frac{1}{n})\)
      🔗
    🔗

    E 0 DNE DNE DNE

    Section Maxima and suprema

    Theorem 28.
    Let \(A \subseteq \mathbb{R}\) be a nonempty set that is bonded above. Let \(s \in \mathbb{R}\) be an uppoer bound for \(A\text{.}\) Then \(s=\sup(A)\) if and only if, for every \(\epsilon > 0\text{,}\) there is an \(a \in A\) with \(s-\epsilon a\text{.}\)
    🔗
    🔗
    We will prove this Lemma next time.
    🔗
    Note 29.
    With this lemma we will have two ways of proving statements about suprema: the definition (to make arguments from first principles), and this lemma (which is an equivalent characterization of suprema).
