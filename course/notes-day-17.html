    Section Compactness

    Definition 194.
    A set \(K \subset \mathbb{R}\) is (sequentially) compact if every sequences in \(K\) has a subsequence that converges in \(K\text{.}\)
    🔗
    🔗
    Theorem 195.
    A set\(K \subset \mathbb{R}\) is (sequentially) compact if and only if \(K\) is closed and bounded.
    🔗
    🔗
    Theorem 196. Heine-Borel.
    A set \(K \subset \mathbb{R}\) is compact if and only if every open cover of \(K\) contains a finite subcover.
    🔗
    🔗
    Note writing ``\(K\) has the Heine-Borel property’’ means that ``every open cover of \(K\) has a finite subcover.’’
    🔗
    Proof.
    (\(\Leftarrow\)) Suppose \(K\) has the property that every open cover of \(K\) has a finite subcover. This is called the Heine-Borel property. We want to show that \(K\) is sequentially compact. We will do this by showing that \(K\) is closed and bounded.
    🔗
    To show that \(K\) is bounded, consider the open sets
    \begin{align*} V_1(x)\amp =\{y \in \mathbb{R}\, : \, |x-y| < 1\} \end{align*}
    These are the \(1-\)neighborhoods of \(x\text{.}\) Then the collection \(\{v_1(x)\}_{x \in K}\) is an open cover of \(K\text{.}\) Since \(K\) has the Heine-Borel property, this open cover has a finite subcover \(\{V_1(x_k)\}_{k=1}^n\text{,}\) noting that \(x_k \in K\) for all \(k\text{.}\)
    🔗
    🔗
    🔗

    Section

    Proof.
    Without loss of generality, we can assume that \(x_i < x_{i+1}\) for \(1=1,2,3,\ldots,N-1\text{.}\) Then
    \begin{align*} x_1 -1 \amp < x < x_n+1 \end{align*}
    so \(K\) is bounded by \(\max\{|x_1-1|,x_{n}+1|\}\text{.}\)
    🔗
    To show that \(K\) is closed we argue indirectly. Suppose \((y_k) \subset K\) is a Cauchy sequence with limit \(y\text{.}\) This means that \(y\) is a limit point of \(K\text{.}\) We need to show \(y \in K\text{.}\)
    🔗
    Suppose not. Suppose \(y \notin K\text{.}\) Then \(|x-y| > 0\) for all \(x \in K\text{.}\) (The inequality equals zero precisely when \(x=y\text{.}\)). Define
    \begin{align*} \mathcal{O}_x \amp =\{u \in K \, : \, |u-x| < \frac{1}{2}|x-y|\} \end{align*}
    🔗
    The closer \(x\) is to \(y\text{,}\) the smaller the radius of \(\mathcal{O}_x\text{.}\) The collection \(\{\mathcal{O}_x\}_{x \in K}\) is an open cover of \(K\text{.}\) By the Heine-Borel property, this open cover has a finite subcover, \(\{\mathcal{O}_{x_i}\}_{i=1}^{n}\text{.}\)
    🔗
    Consider \(\epsilon_0 = \min\{\frac{1}{2}|x_i-y|\}\text{,}\) which is the smallest radius of the open sets in our finite subcover. Since \((y_n) \rightarrow y\text{,}\) there is an \(N > 0\) such that \(|y-n-y| < \epsilon_0\) for \(n > N\text{.}\) But these \(y_n\) are outside every \(\mathcal{O}_{x_i}\text{.}\) So out purported finite subcover does not cover \(K\text{!}\) This is a contradicion.
    🔗
    Consequently, \(K\) is compact.
    🔗
    🔗
    🔗

    Section The Heine-Borel Property

    Example 197. The singleton set has the Heine-Borel property.
    Let \(x \in \mathbb{R}\) and consider \(F=\{x\}\text{,}\) the singleton set.
    🔗
    What would an open cover of \(F\) look like. Consider \(I=(a,b)\) where \(a < x < b b\text{.}\) Then \(x \in I\text{.}\) The collection \(\mathcal{U}=\{I\}\) is an open cover of \(F\text{.}\)
    🔗
    Suppose \(\mathcal{U}=\{U_{\lambda}\,:\, \lambda \in \Lambda }\) is an open cover of \(F\text{.}\) This means
    \begin{align*} F \amp \subset \bigcup_{\lambda \in \Lambda} U_{\lambda}. \end{align*}
    So \(x \in U_{\lambda_0}\) for some \(\lambda_0 \in \Lambda\text{.}\) Let’s define \(\mathcal{U}_0 = \{ U_{\lambda_0}\}\text{.}\) Then \(\mathcal{U}_0 \subset \mathcal{U}\) is a subset of our open cover of \(F\text{.}\) And because \(\mathcal{U}_0\) is a finite collection, it is a finite subcover of \(\mathcal{U}\text{.}\)
    🔗
    Note that we’ve just proved that \(F=\{x\}\) has the Heine-Borel property. That is to say, \(\{x\}\) is a compact set.
    🔗
    🔗
    Example 198. A finite set has the Heine-Borel property.
    Let \(x_i \in \mathbb{R}\) and consider \(F=\{x_1,x_2,x_3,\ldots,x_n\}\text{.}\) Suppose \(\mathcal{U}=\{U_{\lambda}\,:\, \lambda \in \Lambda }\) is an open cover of \(F\text{.}\) This means
    \begin{align*} F \amp \subset \bigcup_{\lambda \in \Lambda} U_{\lambda}. \end{align*}
    Let’s see how to extract a finite subcover of \(F\) from \(\mathcal{U}\text{.}\) Take \(x_1 \in F\text{.}\) Since \(\mathcal{U}\) is an open cover of \(F\text{,}\) \(x_1 \in \bigcup_{\lambda \in \Lambda} U_{\lambda}\text{.}\) This means \(x_1 \in U_{\lambda_1}\) for some \(\lambda_1 \in \Lambda\text{.}\) Set this open set aside for our finite subcover, then consider \(x_2 \in \bigcup_{\lambda \in \Lambda} U_{\lambda}\text{.}\) This means \(x_2 \in U_{\lambda_2}\) for some \(\lambda_2 \in \Lambda\text{.}\) Add this set to our collection of open set, then repeat this process for each \(x_k\) that remains to find \(U_{\lambda_k}\) containing \(x_k\text{.}\) When we exhaust the numbers in \(F\text{,}\) we end up with a collection of at most \(n\) open sets \(\{ U_{\lambda_k} \, : \, 1 \lewq k \leq n\}\text{.}\) This is a finite subcover of \(F\text{.}\)
    🔗
    This proves that \(F\) has the Heine-Borel property. That is to say, \(\{x_1, x_2, x_3, \ldots, x_n\}\) is a compact set.
    🔗
    🔗
    Example 199. The open interval \((0,1)\) does not have the Heine-Borel property.
    Let \(\mathcal{O}=(0,1)\text{.}\) Let
    \begin{align*} \mathcal{U} \amp =\{ (\epsilon,1) \subset \mathbb{R} \, : \, 0 < \epsilon < 1\} \end{align*}
    This is an open cover of \(\mathcal{O}\) because given any \(x \in \mathcal{O}\text{,}\) there is an \(\epsilon > 0\) with \(0 < \epsilon < x\text{.}\) Thus \(x \in (\epsilon,1)\text{,}\) an open set in the collection \(\mathcal{U}\text{.}\)
    🔗
    If \(\mathcal{O}\) had the Heine-Borel property, \(\mathcal{U}\) would have a finite subcover, \(\mathcal{U}_0\text{.}\) But then the intervals in the subcover would have a smallest left endpoint, \(\epsilon_0 > 0\text{.}\) By the Archimedean Property of the reals, this leaves real numbers between 0 and \(\epsilon_0\) uncovered! Thus \(\mathcal{U}_0\) could not be a cover of \((0,1)\text{.}\)
    🔗
    This shows that \(\mathcal{O}\) does not have the Heine-Borel property. That is to say, \(\mathcal{O}\) is not compact.
    🔗
